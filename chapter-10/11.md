# 10. Production envelope와 대화형 Plotly

## 10.0 생산–성장 트레이드오프를 포락선으로 해석하기

세포공장을 설계할 때에는 목표 물질을 더 많이 만들도록 제한했을 때 성장이 얼마나 달라지는지 점검해야 합니다. FBA 한 번은 한 목적함수 아래의 한 최적해만 제시합니다. [Production envelope](../glossary.md)는 선택한 반응을 여러 값에 고정하고, 각 지점에서 **지정한 목적함수**의 최소·최대를 다시 최적화하여 생산–생장 feasible region의 경계를 보여 줍니다. 따라서 어느 반응을 고정했는지와 어느 반응을 목적함수로 두었는지를 함께 읽어야 합니다. 이 개념과 [phenotype phase plane(PhPP)](../glossary.md)의 이론적 배경, 균주 설계에서의 활용은 [Chapter 8](../chapter-8/README.md)에서 자세히 다룹니다.

## 10.1 Acetate 생산 포락선 계산

> **선행 셀 필요: §1.1의 `model`, `configuration`, `results`**

```python
from cobra.flux_analysis import production_envelope

configuration.processes = 1
envelope = production_envelope(
    model,
    reactions=["EX_ac_e"],           # 생산물로 볼 반응: acetate exchange.
    carbon_sources="EX_glc__D_e",    # 탄소원 기준으로 수율을 정규화.
    points=21,                        # acetate flux를 0~최대까지 21개 지점으로 나눔.
)

peak_row = envelope.loc[envelope["flux_maximum"].idxmax()]  # 성장률이 가장 높은 지점.
peak_growth = float(peak_row["flux_maximum"])
acetate_at_peak_growth = float(peak_row["EX_ac_e"])

results["envelope_peak_growth"] = peak_growth
results["envelope_acetate_at_peak_growth"] = acetate_at_peak_growth
results["envelope_max_acetate"] = float(envelope["EX_ac_e"].max())

print(envelope[["EX_ac_e", "flux_minimum", "flux_maximum"]]
      .head().round(6).to_string(index=False))
print("peak growth / acetate at peak:", peak_growth, acetate_at_peak_growth)

assert abs(peak_growth - 0.873921507) < 1e-6
assert abs(acetate_at_peak_growth) < 1e-9
```

기본 `e_coli_core` 조건에서 최대 성장 지점의 acetate secretion은 0이고, acetate를 더 많이 고정할수록 가능한 최대 성장이 감소합니다(표의 `flux_maximum` 열이 `EX_ac_e`가 커질수록 단조 감소합니다). 이 표에서 고정한 반응은 `EX_ac_e`이고 기본 목적함수는 biomass이므로, `flux_minimum`과 `flux_maximum`은 모두 **성장률**입니다. 따라서 이 표의 `flux_minimum=0`은 해당 acetate 고정값에서 가능한 최소 성장률이 0이라는 뜻일 뿐, acetate의 최소 생산량이 0이라는 뜻은 아닙니다.

성장 결합은 반대로 biomass를 고정하고 acetate secretion을 목적함수로 두어 판정합니다. 아래 계산에서 양의 biomass grid의 `flux_minimum`은 그 성장률에서 가능한 **최소 acetate secretion flux**입니다. 이 값이 0인 양의 생장 지점이 하나라도 있으면, 그 고정 조건과 grid에서는 강제 growth coupling을 입증할 수 없습니다.

```python
coupling_envelope = production_envelope(
    model,
    reactions=["Biomass_Ecoli_core"],  # 고정하는 축: 성장률 (h^-1).
    objective="EX_ac_e",               # 최소·최대로 다시 최적화할 값: acetate secretion.
    carbon_sources="EX_glc__D_e",
    points=21,
)
positive_growth = coupling_envelope.query("Biomass_Ecoli_core > 1e-9")
minimum_acetate = positive_growth["flux_minimum"]

assert (minimum_acetate >= -1e-9).all()
assert (minimum_acetate.abs() < 1e-9).any()
print("sampled positive-growth point with zero minimum acetate:",
      float(positive_growth.loc[minimum_acetate.abs().idxmin(), "Biomass_Ecoli_core"]))
```

기본 `textbook` 모델과 위 조건에서 재생성한 21점 grid에는 최소 acetate flux가 0인 양의 생장 지점이 있습니다. 그러므로 이 결과는 성장–생산 trade-off를 보이지만, acetate의 강제 growth coupling을 입증하지는 않습니다. 이는 모델·배지·bounds·grid에 조건부인 계산 결과이며 산업 생산성이나 실험 관측을 뜻하지 않습니다.

{% hint style="warning" %}
**해석상의 주의: "production envelope가 아래로 기울어 있으면 항상 성장 결합(growth-coupled) 균주다"**

아닙니다. 이 절의 첫 envelope는 acetate flux를 늘릴수록 최대 성장이 줄어드는 "트레이드오프"만 보여 줍니다. 여기의 `flux_minimum`은 고정한 acetate 조건에서의 **최소 성장률**이므로 growth coupling 판정에 사용하면 안 됩니다. growth coupling은 biomass를 고정하고 acetate를 목적함수로 다시 계산했을 때의 `flux_minimum`으로 판정합니다. 양의 biomass 지점에서 그 최소 acetate flux가 0이면, 해당 조건에서는 "acetate를 만들지 않고도 가능한 성장"이 남아 있으므로 강제 coupling을 입증하지 못합니다.
{% endhint %}

{% hint style="warning" %}
**흔한 에러**: `carbon_sources` 인자에 실제로 uptake bound가 열려 있지 않은 반응 ID를 주면(예: 이미 `medium`에서 막아 둔 탄소원), `production_envelope`가 모든 지점에서 성장률 0을 반환할 수 있습니다. `model.medium`을 먼저 확인해 지정한 탄소원의 섭취 용량이 0보다 큰지 점검하십시오.
{% endhint %}

## 10.2 Plotly로 hover·zoom·범례 토글하기

> **선행 셀 필요: §10.1의 `envelope`; 선택 사항(`plotly`)**

```python
import plotly.graph_objects as go

fig = go.Figure()
fig.add_trace(
    go.Scatter(
        x=envelope["EX_ac_e"],
        y=envelope["flux_minimum"],
        mode="lines+markers",
        name="minimum growth",
        line=dict(color="#9ecae1"),
        hovertemplate="acetate=%{x:.2f}<br>min growth=%{y:.3f}<extra></extra>",
    )
)
fig.add_trace(
    go.Scatter(
        x=envelope["EX_ac_e"],
        y=envelope["flux_maximum"],
        mode="lines+markers",
        name="maximum growth",
        line=dict(color="#08519c"),
        fill="tonexty",
        fillcolor="rgba(49, 130, 189, 0.20)",
        hovertemplate="acetate=%{x:.2f}<br>max growth=%{y:.3f}<extra></extra>",
    )
)
fig.update_layout(
    title="Acetate production envelope",
    xaxis_title="Acetate secretion flux (mmol gDW⁻¹ h⁻¹)",
    yaxis_title="Growth rate (h⁻¹)",
    template="plotly_white",
    hovermode="x unified",
)
fig.show()
```
