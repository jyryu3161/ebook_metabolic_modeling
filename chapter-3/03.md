# 3. 모델 구획(compartment)

## 3.1 생물학적 공간과 모델 구획

세포 안은 하나의 균일한 공간이 아니다. 세포막, 소기관막, 세포벽과 막 사이 공간, 그리고 시스템 경계가 세포를 여러 칸으로 나눈다. 이렇게 화학적 환경과 물질 접근성이 서로 다른 공간을 생물학에서 [구획(compartment)](../glossary.md)이라 부른다. GEM에서 구획은 대사물 species와 반응의 위치를 구별하는 **모델 label**이다.

두 개념은 관련되지만 동일하지 않다.

- 세포질·미토콘드리아 기질·주변세포질은 실제 생물학적 공간을 나타낸다.
- `e`(extracellular)는 세포 내부 소기관이 아니라 모델과 환경의 접점을 나타내는 시스템 구획이다.
- 하나의 생물학적 공간을 모델 목적에 따라 더 세분하거나, 여러 공간을 하나로 합칠 수 있다.
- 같은 화학종이라도 구획이 다르면 `pyr_c`와 `pyr_m`처럼 서로 다른 대사물 행으로 표현한다.

구획을 분리하면 막을 가로지르는 이동이 자동으로 일어나지 않는다. 서로 다른 구획의 species는 명시적인 [운송 반응](04.md)으로만 연결된다. 이 구조는 막 투과성, 에너지 결합, redox shuttle과 organelle-specific pathway를 모델에 반영한다.

원핵생물을 “구획이 없는 하나의 공간”으로 일반화해서는 안 된다. 그람음성 세균에는 세포질, 내막, 주변세포질, 외막이 있으며, 일부 세균과 고세균은 단백질성 microcompartment나 특수 막 구조를 가진다. GEM의 구획 수는 이러한 생물학을 얼마나 상세히 재구축했는지에 따라 달라진다.

## 3.2 대장균 모델의 구획

동일한 생물 종이라도 모델 범위에 따라 구획 표현이 다르다.

| 모델 | 구획 | 해석 |
|:---|:---|:---|
| `e_coli_core` `textbook` | `c`, `e` | 주변세포질과 외막 통과를 축약한 교육용 모델 |
| iML1515 | `c`, `p`, `e` | 세포질, 주변세포질, 세포외 species를 구분한 재구축 |

*표 3.1. 대장균 모델의 구획 표현. `e`는 세포의 해부학적 내부가 아니라 모델 경계와 연결되는 extracellular compartment이다. 실제 ID와 구획 목록은 고정한 모델 릴리스에서 확인한다. 모델 수치는 [e_coli_core 교육 모델](https://pubmed.ncbi.nlm.nih.gov/26443778/)과 [iML1515, DOI 10.1038/nbt.3956](https://doi.org/10.1038/nbt.3956)을 기준으로 한다.*

그람음성 세균에서 외부 기질은 외막 porin, 주변세포질 결합 단백질, 내막 운송체를 순차적으로 거칠 수 있다. `e_coli_core`는 일부 단계를 하나의 운송 반응으로 축약한다. 이 축약은 계산 오류가 아니라 모델 범위의 선택이지만, 막별 수송 기전을 분석하는 연구에는 적합하지 않을 수 있다.

## 3.3 진핵생물 모델의 구획

인체 GEM은 세포질과 세포외 공간 외에 여러 소기관을 구분한다. 대표적 위치와 대사 기능은 다음과 같다.

| 모델 구획 | 대표 기능 | 모델링상의 핵심 |
|:---|:---|:---|
| 세포질 `c` | 해당과정, pentose phosphate pathway, 일부 지질 합성 | 다른 소기관과 다수의 운송 반응으로 연결 |
| 미토콘드리아 `m` | TCA cycle, 호흡, 일부 지방산 산화 | 내막 운송과 전기화학적 결합이 중요 |
| 소포체 `r` | 지질·스테롤 대사, 단백질 가공 관련 대사 | lumen과 막의 구분 수준은 모델별로 다름 |
| 골지체 `g` | glycosylation과 분비 경로 관련 대사 | 소포체·세포질과의 물질 전달 필요 |
| 리소좀 `l` | 고분자·복합지질 분해 | acidic lumen의 반응 방향과 수송이 중요 |
| 퍼옥시좀 `x` | 매우 긴 사슬 지방산 산화, 과산화물 대사 | 미토콘드리아·세포질과의 shuttle 필요 |
| 핵 `n` | 일부 nucleotide·transport 과정 | 핵공을 통한 이동의 상세도는 모델별로 다름 |
| 세포외 `e` | 환경·체액과의 경계 species | exchange reaction과 연결 |

*Table 3.2. 인체 GEM에서 흔히 사용하는 구획의 기능적 해석. 실제 구획 ID와 포함 범위는 Recon3D, Human-GEM 등 모델 릴리스별 compartment 목록을 확인해야 한다.*

pH, NAD$$^+$$/NADH, NADP$$^+$$/NADPH, ATP/ADP 비율은 조직, 영양 상태, 측정법과 세포 상태에 따라 달라진다. 단일 비율을 “세포질의 고정값” 또는 “미토콘드리아의 고정값”으로 제시할 수 없다. 표준 화학량론 모델의 $$\mathbf{S}$$도 이러한 농도비를 직접 저장하지 않는다. 열역학적 방향성이나 redox 상태를 정량 분석하려면 조건별 농도·활동도와 Gibbs energy 자료를 별도 제약으로 통합해야 한다.

## 3.4 직사각형 화학량론 행렬의 구획 블록

[화학량론 행렬](../chapter-2/README.md)은 대사물 행과 반응 열의 수가 다른 직사각형 행렬이다. 따라서 구획 구조를 동일 노드 집합의 인접행렬처럼 “대각 블록=구획, 비대각 블록=운송”으로 해석하면 안 된다.

두 구획 `c`와 `m`의 대사물 행을 각각 $$M_c,M_m$$, 내부 반응 열을 $$R_c,R_m$$, 두 구획을 잇는 운송 반응 열을 $$R_T$$로 묶으면 열 순서 $$(R_c,R_m,R_T)$$에 대해

$$
\mathbf{S}
=
\begin{bmatrix}
\mathbf{S}_{c} & \mathbf{0} & \mathbf{T}_{c}\\
\mathbf{0} & \mathbf{S}_{m} & \mathbf{T}_{m}
\end{bmatrix}
$$

로 배열할 수 있다.

- $$\mathbf{S}_c$$와 $$\mathbf{S}_m$$은 각 구획 내부 반응 열의 계수이다.
- 동일한 운송 반응 열 집합 $$R_T$$가 두 행 블록에 걸쳐 나타난다.
- 한 운송 반응이 `pyr_c`를 소비하고 `pyr_m`을 생성하면 해당 열은 $$\mathbf{T}_c$$에 음수, $$\mathbf{T}_m$$에 양수 계수를 가진다.
- exchange reaction 열은 보통 extracellular row block에 하나의 비영 계수를 가진다.

세 구획 이상에서도 원리는 같다. 행은 구획별 species로 분할하고, 열은 구획 내부 반응·구획 간 운송·경계 반응으로 분할한다. 운송 반응은 “비대각 정사각 블록”이 아니라 여러 행 블록에 동시에 비영 원소를 갖는 **열**이다.

이 배열은 시각적·계산적 편의를 위한 순열이다. 행과 열의 순서를 바꾸어도 모델의 물질수지와 feasible flux set은 변하지 않는다.

## 3.5 구획 배정의 근거

반응을 특정 구획에 배정하려면 촉매 단백질과 기질·생성물의 위치 근거가 필요하다. 근거의 우선순위는 연구 목적에 따라 달라질 수 있으나 일반적으로 다음 자료를 조합한다.

1. 직접적인 효소 활성·현미경·분획 실험
2. organelle-resolved proteomics와 신뢰도 높은 curated database
3. 수송 peptide, transmembrane segment, targeting motif에 대한 sequence-based prediction
4. 상동 단백질과 경로 일관성
5. gap-filling 또는 모델 가능성만을 위한 잠정 배정

잠정 배정은 근거 코드(evidence code)와 함께 표시하고 민감도 분석 대상으로 남겨야 한다. 재구축 과정에서 위치 근거를 문헌·데이터베이스와 교차 검증하는 원칙은 Thiele and Palsson(2010)의 프로토콜에 제시되어 있다([DOI: 10.1038/nprot.2009.203](https://doi.org/10.1038/nprot.2009.203), [PMC3125167](https://pmc.ncbi.nlm.nih.gov/articles/PMC3125167/)).

## 3.6 Localization benchmark의 비교 규약

서로 다른 논문에서 보고한 “정확도 90%”와 “정확도 85%”를 직접 순위화할 수는 없다. 다음 조건이 같지 않으면 수치 차이가 알고리즘이 아니라 benchmark 구성에서 발생할 수 있다.

- 도구와 모델 버전
- 대상 생물군과 단일·다중 label 설정
- 구획 class의 정의와 개수
- train/test sequence identity cutoff
- 독립 test set 사용 여부
- class imbalance
- accuracy, macro-F1, MCC, precision, recall 가운데 사용한 지표
- dual-localized protein의 채점 방식

비교표에는 단일 “정확도” 대신 다음 항목을 기록한다.

| 필수 보고 항목 | 예 |
|:---|:---|
| 도구·버전 | DeepLoc 2.0 |
| 예측 문제 | 10개 구획 multi-label |
| 데이터셋 | 논문 독립 test set 또는 사용자 외부 검증셋 |
| 중복 제거 | sequence identity cutoff |
| 지표 | class별 precision/recall, macro-F1, MCC |
| 불확실성 | bootstrap confidence interval 또는 반복 분할 |

DeepLoc 2.0은 protein language model을 사용하는 multi-label localization predictor의 한 예이며, 논문의 보고 수치는 해당 데이터셋과 평가 설계 안에서만 해석해야 한다([DOI: 10.1093/nar/gkac278](https://doi.org/10.1093/nar/gkac278), [PMC9252801](https://pmc.ncbi.nlm.nih.gov/articles/PMC9252801/)).

두 구획 분류에서 혼동행렬을 사용한다면

$$
\mathrm{precision}
=
\frac{TP}{TP+FP},
\qquad
\mathrm{recall}
=
\frac{TP}{TP+FN}
$$

을 구분한다. 불균형 다중 class 문제에서는 전체 accuracy 하나보다 class별 지표와 macro average가 더 적절할 수 있다.

## 3.7 구획 수와 운송 반응 수

구획이 $$k$$개라는 사실만으로 운송 반응 수를 $$\binom{k}{2}$$ 또는 $$k-1$$로 계산할 수 없다. 실제 운송 반응 수는 다음에 의해 결정된다.

- 실제로 접촉하거나 물질을 교환하는 구획 쌍
- 각 경계를 통과하는 화학종 수
- uniport·symport·antiport와 에너지 결합의 구분
- 같은 화학종에 대한 여러 운송체와 GPR
- 수송 방향과 조건별 bounds

구획 연결 그래프의 가능한 쌍 수는 위상적 상한을 제공할 수 있지만, GEM 반응 수의 예측식은 아니다. 실제 모델 비교에서는 transport reaction을 동일한 판정 규칙으로 직접 집계해야 한다.

---

### 해석상의 주의

- 모델 구획 수는 생물체의 고정된 해부학적 구획 수가 아니라 재구축 상세도의 속성이다.
- extracellular compartment는 세포 내부 소기관이 아니다.
- 위치별 pH와 redox ratio는 조건 의존적이며 표준 $$\mathbf{S}$$에 직접 저장되지 않는다.
- 직사각형 $$\mathbf{S}$$에서 운송은 여러 구획 행 블록에 비영 계수를 갖는 반응 열로 표현된다.
- localization 도구 성능은 동일 benchmark와 동일 지표에서만 직접 비교한다.
