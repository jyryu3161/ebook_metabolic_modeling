# 7. 상태 변환법과 개인화 GEM의 질병 응용

§6에서 MTA/rMTA라는 하나의 구체적인 알고리즘을 자세히 배웠으니, "질병 대사 모델링 = MTA"라고 여기기 쉽습니다. 그러나 이 절에서 살펴볼 다섯 가지 질병 사례(암·당뇨병·NAFLD·노화·COVID-19)를 미리 훑어보면, 실제로 MTA/rMTA의 완전한 형태를 사용한 것은 §7.4(노화)와 §7.5(COVID-19)뿐이고, §7.1(암)은 PRIME이라는 별개의 방법을, §7.2(당뇨병)는 GIMME 기반 다중 조직 통합을, §7.3(NAFLD)은 순수 발현 비교와 후속 동물 실험을 사용했습니다. 즉 "상태를 변환한다"는 큰 아이디어는 공유하지만, 그 아이디어를 계산으로 구현하는 구체적인 방법은 사례마다 다릅니다.

이 절의 사례가 모두 MTA/rMTA를 사용한 것은 아닙니다. PRIME·조직별 FBA·대사 작업 분석은 “세포 특이적 모델을 만든다”는 공통점이 있지만 목적함수와 출력이 다릅니다. 논문을 읽을 때 **모델 추출법, flux 시뮬레이션법, 표적 순위화법**을 분리해 기록하십시오.

| 절 | 질병 | 모델 추출법 | flux 시뮬레이션/순위화법 | MTA/rMTA 사용 여부 |
|---|---|---|---|---|
| §7.1 | 암(NCI-60) | PRIME | 세포주별 성장·선택성 분석 | 아니오 |
| §7.2 | 제2형 당뇨병 | GIMME(조직별 발현 통합) | 다중 조직 전신 모델 FBA | 아니오 |
| §7.3 | NAFLD | iHepatocytes2322(발현 비교) | 대사 작업 분석 + 동물실험 | 아니오 |
| §7.4 | 노화(효모) | iMAT | rMTA 이전의 원 MTA | **예(원 MTA)** |
| §7.5 | COVID-19 | iMAT + flux sampling | rMTA | **예(rMTA)** |

이 표가 주는 실천적 교훈은 분명합니다 — 논문 제목에 "대사 정상화"나 "상태 변환"이라는 표현이 등장한다고 해서 자동으로 MTA를 썼다고 가정해서는 안 되며, Methods 절을 직접 확인해 어떤 알고리즘이 실제로 쓰였는지 확인해야 합니다.

## 7.1 암: MLYCD

Yizhak et al. (2014)의 **PRIME** 연구는 NCI-60과 HapMap 세포 특이적 모델에서 **MLYCD(Malonyl-CoA Decarboxylase)** 같은 암 대사 취약성을 제안했습니다. PRIME은 MTA가 아니라 발현으로 개인화한 모델의 성장·선택성 분석입니다. MLYCD 억제는 말론일-CoA–CPT1–지방산 산화 축을 바꾸는 가설로 후속 실험되었지만, 관찰된 세포주 결과를 모든 암과 정상 조직의 선택성으로 일반화해서는 안 됩니다.

이 사례는 §3~§5에서 배운 "죽이기" 접근과 §6의 "고치기" 접근이 실제로는 같은 저자·같은 연구 계열(Yizhak, Ruppin 등)에서 나란히 발전했다는 점에서도 흥미롭습니다. §1.2의 AFR과 §5.3의 반-이동성 표적을 제안한 PRIME이 여기서는 MLYCD라는 또 다른 성장 취약성을 제시하는 데도 쓰였습니다 — 같은 세포 특이적 모델 구축 기법이 "이동성 억제"와 "증식 억제"라는 서로 다른 질문에 재사용될 수 있음을 보여줍니다.

## 7.2 제2형 당뇨병: 다중 조직 모델링

Bordbar et al. (2011)의 간·지방·골격근 통합 전신 모델에 GIMME로 조직별 발현 데이터를 통합한 결과(Mardinoglu et al. 2016), T2D 환자 간 모델에서 젖산·피루베이트 기원 당신생합성 증가, NAFLD 환자에서 글리세롤의 당신생합성 기질화, 대사 유연성(metabolic flexibility) 저하가 확인되었습니다. BCAT1 표적, 미토콘드리아 기능 회복, 세린 결핍 보충이 치료적 시사점으로 제시되었습니다.

이 사례가 §1.6의 Randle cycle 논의와 다시 만난다는 점에 주목하십시오. 간·지방·근육 세 조직을 하나의 화학량론 시스템으로 통합한 이 모델은, 한 조직(지방)에서 방출된 유리지방산이 다른 조직(간·근육)의 대사에 미치는 파급 효과를 직접 계산할 수 있다는 점에서 §7.1의 단일 세포주 모델과는 근본적으로 다른 스케일의 질문에 답합니다. "정상화 표적"이 반드시 병든 조직 안에 있어야 하는 것은 아니라는 점 — 예를 들어 간의 문제를 지방 조직의 지방분해를 조절해 완화할 수도 있다는 가능성 — 이 다중 조직 모델링의 독특한 기여입니다.

## 7.3 NAFLD: 세린 합성 경로

§1.6에서 소개한 iHepatocytes2322 모델의 세린 결핍 발견은 **PSPH 활성화**, **SHMT1 활성화**를 잠재적 정상화 표적으로 제시했습니다. 고지방식이(HFD) 동물 모델에서의 세린 보충은 간 중성지방 축적·염증·간효소(ALT/AST) 수치를 개선하여 대사 정상화 가설을 뒷받침했습니다.

이 사례는 §7의 모든 사례 중 "정상화 개입"이 가장 직관적인 경우입니다 — MTA처럼 복잡한 MIQP를 풀 필요 없이, "부족한 것을 보충한다"는 단순한 논리(세린 결핍 → 세린 보충)만으로도 이미 하나의 정상화 가설이 됩니다. 이는 §1.6의 "잠깐, 생각해보기" 상자에서 논의했듯, 어떤 질병은 발현이 낮아진 유전자를 다시 끌어올리는 방향(활성화)의 개입이 필요하다는 것을 실제 동물 실험 데이터로 보여준 예이기도 합니다.

## 7.4 노화: 효모 수명 연장 (GRE3, ADH2)

MTA 원 논문(Yizhak et al., 2013)은 노화 효모 전사체를 source로, 칼로리 제한과 관련된 장수 상태를 target으로 두고 상위 7개 결손 후보를 시험했습니다. **GRE3**와 **ADH2** 결손이 chronological lifespan을 유의하게 늘렸고, GRE3 결손은 중앙 수명을 약 두 배로 늘렸습니다. 무작위 효모 유전자 중 장수 결손의 기대 빈도는 약 3.5%로 추정되어, 2/7의 성공은 약 10배 enrichment였습니다. 두 결손에서 약한 ROS 증가가 관찰되어 hormesis 기전이 제안되었지만, 이는 “부분 억제가 완전 결손보다 낫다”는 실험 결과를 뜻하지 않습니다.

{% hint style="success" %}
✏️ **enrichment 배수를 직접 확인해 봅시다.** 무작위로 유전자를 골랐을 때 장수 결손일 확률이 3.5%(=0.035)라면, 7개를 무작위로 뽑았을 때 기대되는 성공 개수는 $$7\times0.035\approx0.245$$개입니다. 실제로는 2개(GRE3, ADH2)가 성공했으므로, enrichment 배수는

$$\frac{2/7}{0.035}=\frac{0.286}{0.035}\approx8.2\text{배}$$

본문에서 "약 10배"라 한 것은 반올림된 근사치입니다. 이 계산이 왜 중요할까요? "7개 중 2개가 성공했다"는 사실 그 자체는 성공률이 낮아(29%) 인상적이지 않아 보일 수 있지만, "무작위로 골랐다면 0.245개 정도만 성공했을 것"이라는 기준선과 비교하면 이 결과가 우연이라 보기 어려울 만큼 높은 성공률이라는 것이 드러납니다. 이것이 바로 §3.6에서 배운 "무작위 기준선과 비교하라"는 원칙이 표적 발굴 이외의 맥락(수명 연장 스크리닝)에서도 똑같이 적용되는 예입니다.
{% endhint %}

이 사례에서 §1.6의 "노화 = 대사 재프로그래밍의 반대 극단"이라는 통찰이 실제로 계산에 반영되는 방식에 주목하십시오 — source는 "늙은" 상태, target은 "칼로리 제한으로 젊어진" 상태이며, MTA는 이 방향으로 세포를 되돌릴 개입을 찾습니다. 암 표적 발굴(source=암, target=정상)과 정확히 같은 수학적 틀이지만, 이번에는 "화살표의 방향"이 노화에서 젊음으로 향한다는 점만 다릅니다.

## 7.5 COVID-19: 숙주-바이러스 공통 표적과 약물 재배치

> **핵심 개념·용어(English):** 약물 재배치/재창출(Drug Repositioning/Repurposing) — 기존에 승인된 약물을 새로운 적응증에 적용하는 전략. 대사 정상화 표적이 이미 알려진 약물의 표적과 겹칠 경우, 개발 기간을 크게 단축할 수 있습니다.

한 SARS-CoV-2 연구는 12개 공개 발현 데이터셋의 감염 대 비감염 차이를 분석하고, iMAT·sampling으로 감염 상태의 기준 flux를 만든 뒤 rMTA를 적용했습니다. 특히 remdesivir 처리 감염 세포가 대조 상태 쪽으로 더 이동하도록 만들 후보 반응을 예측하고, 알려진 항바이러스 표적·약물 정보와 교차 비교했습니다. 이 결과는 감염 질환에도 상태 변환 순위화를 적용할 수 있음을 보였지만, rMTA 상위 순위 자체가 바이러스 억제의 실험적 인과 증거는 아닙니다.

## 7.6 실험적 검증과 임상 전환

MTA 예측은 계산적 가설일 뿐이므로, 다음과 같은 단계적 검증이 필요합니다.

1. **계산적 신뢰도 평가**: 교차 검증(LOOCV/K-fold), 퍼뮤테이션 검정, 알려진 표적과의 enrichment(ROC-AUC)
2. **시험관 내 검증**: siRNA(일시적 knockdown, 70~95% 효율) 또는 CRISPR-Cas9(영구적 KO) — MLYCD 사례는 3개 독립 siRNA로 >80% knockdown, 암 세포에서 40~60% 증식 감소를 확인
3. **동물 모델 검증**: 효모(chronological/replicative lifespan), 설치류 질환 모델(db/db, ob/ob, HFD 등)
4. **임상 전환 가능성 평가**: druggability, DrugBank 매칭, essentiality 기반 안전성 평가

```
Computational Prediction → In Vitro (siRNA/CRISPR) → Animal Model → IND → Phase I–III → FDA Approval
        (MTA)                                                        (신약신청)
```

MTA의 궁극적 지향점은 **정밀의학(precision medicine)**입니다 — 환자 개인의 종양/질병 전사체를 입력으로 받아 환자 특이적 정상화 표적을 예측하고, 동반진단(companion diagnostic)과 결합해 치료 반응을 사전에 예측하는 것입니다. 다만 복합 요법의 필요성, 질병의 동적 변화, 개인차, 비용 문제는 여전히 해결 과제로 남아 있습니다.

{% hint style="warning" %}
❓ **잠깐, 생각해보기:** 위 4단계 검증 사다리에서, 만약 어떤 표적이 1단계(계산적 신뢰도)는 통과했지만 2단계(시험관 내)에서 예상과 다른 결과가 나왔다면, 이는 "MTA가 틀렸다"는 뜻일까요?

꼭 그렇지는 않습니다. 실패의 원인은 여러 곳에 있을 수 있습니다 — (1) MTA의 가정(§6.1의 세 가지 가정) 중 하나가 이 특정 세포·조건에서 성립하지 않았을 수도 있고, (2) source/target 전사체의 통계적 분류(§6.3의 $$R_S/R_F/R_B$$)에 잡음이 섞였을 수도 있고, (3) 시험관 내 실험 자체의 조건(세포주, 배지, 시간)이 계산에서 가정한 조건과 어긋났을 수도 있습니다. 계산적 예측과 실험 결과가 불일치할 때는 "모델이 틀렸다"고 성급히 결론짓기보다, 이 세 가지(모델 가정, 분류 잡음, 실험 조건) 중 어디서 어긋났는지를 체계적으로 되짚어보는 것이 생산적입니다.
{% endhint %}

{% hint style="info" %}
📌 **§7 미니 요약 및 이 장 전체 마무리.** 이 절에서는 MTA/rMTA를 포함한 "상태 변환" 아이디어가 암·당뇨병·NAFLD·노화·COVID-19라는 서로 완전히 다른 질병에 어떻게 적용되었는지 살펴보았습니다. 핵심은 각 사례가 사용한 구체적인 방법이 다르다는 것을 인지하는 것(§7 도입부 표), 그리고 어떤 계산적 예측이든 계산적 신뢰도 평가 → 시험관 내 → 동물 모델 → 임상이라는 동일한 검증 사다리를 거쳐야 한다는 것입니다. 이것으로 이 장(Chapter 7) 전체 — "죽이기"(§1,3~5)와 "고치기"(§1.5~6,7)라는 두 갈래의 질병 대사 모델링 전략 — 를 마칩니다. [요약 페이지](summary.md)에서 전체 내용을 한 번 더 정리하고, [실습](lab.md)에서 직접 코드로 확인해 보십시오.
{% endhint %}
