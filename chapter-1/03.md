# 3. 게놈 규모 대사 모델의 구성과 가정

게놈 규모 대사 모델(genome-scale metabolic model, GEM)은 유전체 주석과 생화학적 증거를 바탕으로 한 대사 네트워크 재구축을 수학적 계산에 사용할 수 있도록 형식화한 모델이다. “게놈 규모”는 모든 세포 반응을 완전하게 포함한다는 뜻이 아니라, 대상 생물의 유전체가 지지하는 대사 능력을 체계적이고 추적 가능한 방식으로 포괄하려는 범위를 뜻한다. 주석되지 않은 효소, 자발 반응, 유전자 연관이 알려지지 않은 반응 및 실험적으로 보완된 gap-filling 반응도 모델에 포함될 수 있다.

## 3.1 재구축과 조건부 계산 모델

GEM 연구에서는 **[재구축(reconstruction)](../glossary.md)**과 **[계산 모델(model)](../glossary.md)**을 구분한다.

- 재구축은 반응, 대사물, 유전자 연관, 구획 및 근거 문헌을 모은 생물종 특이적 지식베이스이다.
- 계산 모델은 재구축에 반응 경계조건, 배지, 대사 작업 또는 목적함수와 같은 분석 조건을 추가한 것이다.

따라서 하나의 재구축으로 여러 계산 모델을 만들 수 있다. 동일한 *E. coli* 재구축이라도 포도당 최소배지와 아세테이트 최소배지, 호기와 혐기, 성장 최대화와 ATP 유지 최대화 조건은 서로 다른 가능 영역과 예측을 만든다. 반대로 목적함수를 지정하지 않은 재구축도 지식베이스로서 학술적 가치를 갖는다.

GEM의 주요 구성 요소는 다음과 같다.

| 구성 요소 | 수학·데이터 표현 | 기능 |
|:---|:---|:---|
| 대사물과 반응 | 대사물 집합 $$M$$, 반응 집합 $$R$$ | 모델 경계 안의 화학종과 변환을 정의 |
| [화학량론](../glossary.md) | $$\mathbf{S}\in\mathbb{R}^{m\times n}$$ | 각 반응의 소비·생성 몰비와 물질수지를 표현 |
| [구획](../glossary.md)과 수송 | 구획별 대사물 ID, 수송 반응 | 세포 내 공간 분리와 막 사이 이동을 표현 |
| 반응 경계 | $$\mathbf{l}\leq\mathbf{v}\leq\mathbf{u}$$ | 방향성, 환경 공급 및 용량에 관한 허용 범위를 표현 |
| 유전자–단백질–반응 연관 | [GPR](../chapter-3/README.md) 불리언 규칙 | 유전자 교란을 반응 가용성 변화로 변환 |
| 경계·의사반응 | [exchange](../glossary.md), [demand](../glossary.md), [sink](../glossary.md), [biomass](../glossary.md), [ATP maintenance](../glossary.md) | 환경 교환과 모델링 목적의 집계 요구량을 표현 |
| 주석과 근거 | 데이터베이스 ID, 유전자 좌위, 참고문헌, confidence note | 반응의 출처와 큐레이션 판단을 추적 |
| 분석 조건 | 배지, 목적 벡터 $$\mathbf{c}$$, 대사 작업 | 특정 연구 질문에 대한 계산 문제를 정의 |

이 목록은 “완전한 GEM”을 판정하는 고정 체크리스트가 아니다. 예를 들어 성장 예측에는 바이오매스 반응이 필요하지만, 네트워크의 화학량론적 일관성이나 특정 대사물 생산 가능성은 바이오매스 목적함수 없이도 분석할 수 있다. GPR이 없는 반응 역시 자발 반응, 수송·경계 반응, 효소가 미확인된 반응 또는 주석 누락 등 서로 다른 원인을 가질 수 있다.

이 책에서 사용하는 [COBRApy](https://opencobra.github.io/cobrapy/) `textbook` 모델은 GEM의 계산 구조를 보존한 교육용 중심탄소대사 모델이다.

| 항목 | `textbook` 모델에서 확인할 내용 |
|:---|:---|
| 화학량론 행렬 | 구획별 대사물 72개 × 반응 95개 |
| 유전자 객체 | 137개; 모든 반응에 GPR이 존재하는 것은 아님 |
| 구획 | 세포질 `c`와 세포외 `e` |
| 성장 반응 | `Biomass_Ecoli_core` 계열 반응 |
| 환경 조건 | 교환 반응의 하한·상한으로 정의되는 배지 |

이 모델은 전체 *E. coli* 대사를 포괄하지 않으므로 iML1515와 같은 게놈 규모 재구축의 생물학적 완전성을 평가하는 자료로 사용해서는 안 된다. 작은 모델의 목적은 행렬과 경계조건, GPR 및 최적화 결과를 재현 가능한 계산으로 검토하는 데 있다.

## 3.2 증거 기반 재구축

재구축은 다음 증거를 통합하는 반복적 큐레이션 과정이다.

1. **유전체 주석**은 후보 효소 기능과 유전자 좌위를 제시한다.
2. **생화학 데이터베이스**는 반응식, 대사물 식별자, 보조인자 및 가능한 방향성을 제공한다.
3. **일차 문헌과 생물종 특이적 실험**은 후보 반응이 대상 생물에서 작동하는지 판정하는 근거를 제공한다.
4. **성장·기질 이용·분비·[유전자 필수성](../glossary.md) 자료**는 네트워크가 알려진 표현형을 재현하는지 검증한다.
5. **품질 검사**는 원소·전하 균형, [화학량론적 일관성](../glossary.md), 에너지 생성 순환, 주석 완전성 및 모델 버전을 평가한다.

```mermaid
flowchart LR
    GEN["유전체·기능 주석"] --> DRAFT["초안 재구축"]
    DB["반응·대사물 데이터베이스"] --> DRAFT
    LIT["일차 문헌·생물종 특이 증거"] --> CUR["수동 큐레이션"]
    DRAFT --> CUR
    CUR --> QC["질량·전하·연결성·에너지 QC"]
    QC --> MODEL["조건·배지·목적을 부여한 모델"]
    MODEL --> VAL["표현형·플럭스·필수성 검증"]
    VAL -->|불일치 분석| CUR
```

*그림 1.5. 재구축, 계산 모델 형식화 및 실험 검증의 반복 관계. 저자 작성; 개념 근거: Thiele & Palsson (2010), [doi:10.1038/nprot.2009.203](https://doi.org/10.1038/nprot.2009.203).*

재구축과 모델링은 대립되는 용어가 아니라 연속된 단계이다. 재구축은 무엇을 알고 있으며 그 근거가 무엇인지 기록하고, 모델링은 어떤 조건과 가정에서 무엇을 계산할지 정의한다. 유전체 주석과 데이터베이스에서 곧바로 모은 **초안 재구축(draft reconstruction)**은 잘못 포함된 반응(false positive)과 누락된 반응(false negative)을 함께 가질 수 있는 후보 목록이며, 이후 생물종 특이적 증거로 검증·보완한 결과가 **큐레이션된 재구축(curated reconstruction)**이다. 자동화 도구는 초안 생성과 일관성 검사를 가속하지만, 반응 포함 여부와 방향성, 생물종 특이적 기질 이용 및 gap-filling의 생물학적 타당성에는 큐레이션 판단이 필요하다. 구체적인 재구축·품질관리 절차는 [Chapter 5](../chapter-5/README.md)에서 다룬다.

## 3.3 의사-정상 상태 가정

반응 속도와 대사물 양의 단위를 일치시키면 일반적인 물질수지는 다음과 같이 쓸 수 있다.

$$
\frac{d\mathbf{n}}{dt}=\mathbf{S}\mathbf{r}
$$

$$\mathbf{n}$$은 내부 대사물의 양 벡터, $$\mathbf{r}$$은 단위 시간당 반응 진행량 벡터이다. 세포 질량으로 정규화한 특정 플럭스 $$\mathbf{v}$$를 사용할 때에는 세포 성장에 따른 정규화 기준의 변화와 대사물 희석을 엄밀한 동역학식에서 별도로 고려해야 한다.

표준 FBA는 관심 시간 척도에서 내부 소분자 풀이 유의하게 순축적되지 않는다고 근사하여 다음 제약을 사용한다.

$$
\mathbf{S}\mathbf{v}=\mathbf{0}
$$

이를 **[의사-정상 상태 가정](../glossary.md)(pseudo-steady-state assumption)**이라 한다. 이 식은 다음과 같이 해석해야 한다.

- 각 내부 대사물에 대한 총 생성 속도와 총 소비 속도가 일치한다.
- 개별 반응 플럭스는 0이 아닐 수 있으며, 열역학적 평형을 가정하지 않는다.
- 플럭스가 빠르게 “평형에 도달한다”는 진술이 아니라, 분석 시간 구간에서 내부 풀의 순축적을 무시한다는 모델링 근사이다.
- 성장하는 세포에서 거대분자 전구체와 에너지의 순수요는 바이오매스·유지 반응과 같은 drain으로 모델에 포함한다.

화학평형에서는 모든 독립 반응의 순 구동력과 순속도가 사라지지만, 정상 상태에서는 외부 기질을 지속적으로 흡수하고 부산물을 배출하면서 비영 플럭스를 유지할 수 있다. 두 개념을 혼동하면 FBA의 개방계 해석이 무너진다.

### 적용 범위

의사-정상 상태 근사는 일정한 배지에서의 준정상 성장이나, 느리게 변하는 환경을 짧은 구간으로 나눈 분석에 주로 사용한다. 다음 상황에서는 순축적 또는 동역학을 명시적으로 고려해야 한다.

- 영양분의 급격한 공급·고갈 직후와 같은 과도 상태
- 저장 고분자 또는 대사물 풀의 축적·소모가 연구 대상인 경우
- 세포주기 또는 일주기 변화처럼 상태가 시간에 따라 조직적으로 변하는 경우
- 약물 투여 직후의 농도–반응 관계와 효소 포화 동역학을 분석하는 경우

[동적 FBA](../glossary.md)는 세포외 환경을 시간에 따라 갱신하면서 각 시간 구간에서 [FBA](../chapter-4/README.md)를 반복하지만, 내부 대사물 동역학을 완전한 효소 속도식으로 대체하는 방법은 아니다. 동역학 모델과의 차이는 [§4](04.md)에서 다룬다.

## 3.4 GEM이 제공하는 예측과 제공하지 않는 예측

| 분석 결과 | 표준 GEM/FBA가 제공하는 것 | 추가 자료 또는 방법이 필요한 것 |
|:---|:---|:---|
| 성장 | 명시된 바이오매스 반응의 최대 가능 플럭스 | 실제 성장률의 무조건적 보장 |
| 내부 반응 | 제약을 만족하는 플럭스 값 또는 허용 범위 | 대사물 농도, 효소량, 조절 상태 |
| 유전자 결손 | GPR을 통해 제거된 반응 아래의 조건부 성장 예측 | 전사 보상, 독성, 조절 적응의 자동 예측 |
| 분비·섭취 | 경계조건과 목적 아래 가능한 교환 플럭스 | 배지에 대한 시간별 농도 궤적 |
| 경로 사용 | [FVA](../glossary.md)·sampling 등을 통한 가능 영역 분석 | 솔버가 반환한 한 해가 실제 세포의 유일한 상태라는 결론 |

FBA의 표준 정식화와 해석은 Orth, Thiele & Palsson (2010), [doi:10.1038/nbt.1614](https://doi.org/10.1038/nbt.1614)를 따른다. 화학량론 행렬과 정상 상태 제약의 유도는 [Chapter 2](../chapter-2/README.md), GPR·구획·경계·바이오매스 반응의 구현은 [Chapter 3](../chapter-3/README.md)에서 상세히 전개한다.
