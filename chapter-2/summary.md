# 마무리: 요약 · 스스로 점검 · 용어

## 한 장 요약

- 대사 네트워크는 **반응**과 **대사물**이라는 두 종류의 개체와 그 관계로 구성되며, 각 반응은 화학량론·방향성(가역성)·통량 범위·촉매 효소(GPR)라는 속성을 가진다.
- 반응의 방향성은 통량 하한/상한($$v^{lb}, v^{ub}$$)으로 코딩되며, 그 근거는 표준 깁스 자유 에너지 변화의 부호다.
- 대사물 ID의 구획 접미사(`_c`, `_e`, `_m` 등)는 같은 분자라도 위치가 다르면 서로 다른 노드로 다뤄야 함을 나타낸다.
- **화학량론 행렬** $$\mathbf{S} \in \mathbb{R}^{m\times n}$$은 모든 반응의 화학량론 계수를 하나로 압축한 행렬이며, 2~3개 반응짜리 장난감 네트워크를 손으로 직접 구성해 봄으로써 그 구성 규칙을 확인했다.
- 실제 GEM에서 $$\mathbf{S}$$는 매우 희소한 구조를 가진다(*E. coli* core 모델은 5.26%). 이 희소성 이면에는 ATP, NADH 같은 소수의 허브 대사물이 존재한다.
- 대사 네트워크는 이 행렬과 동등하게 **이분 그래프**로도 표현할 수 있으며, 두 표현은 서로 다른 문제(위상 분석 vs. 선형대수/최적화)에 각각 유용하다. 대사물·반응의 연결 정도(degree)를 두 방향(행/열)에서 세어도 항상 같은 값(핸드셰이크 관계)에 도달하며, 영공간의 기저 벡터 하나하나가 그래프 위의 독립적인 경로 하나에 정확히 대응한다는 것을 장난감 네트워크에서 직접 확인했다.
- $$d\mathbf{x}/dt = \mathbf{S}\mathbf{v}$$에서 출발하여 **의사-정상 상태 가정**을 도입하면 $$\mathbf{S}\mathbf{v} = \mathbf{0}$$을 얻는다. 장난감 네트워크를 손으로 풀어 본 결과, 닫힌 네트워크(교환 반응 없음)는 오직 $$\mathbf{v}=\mathbf{0}$$만 허용하지만, 출입구를 열면(교환 반응 추가) 무한히 많은 정상 상태 해가 나타난다.
- 진짜 자유도는 naive한 $$n-m$$이 아니라 $$\mathbf{S}$$의 계수(rank) $$r$$을 반영한 $$n-r$$이다. $$\mathbf{S}$$의 네 가지 기본 부분공간(영공간, 왼쪽 영공간, 열공간, 행공간)은 각각 정상 상태 통량 공간과 보존 화기(예: ATP+ADP+AMP)라는 생물학적으로 의미 있는 구조에 대응하며, *E. coli* core 모델에서 이를 COBRApy로 직접 검증했다($$n-r=28$$, $$m-r=5$$). 교환 반응이 있으면 관련 보존 화기가 깨진다는 것도 열린 장난감 네트워크에서 직접 확인했다($$m-r$$이 1에서 0으로 감소).
- 등식 제약(영공간)에 비가역성의 부호 제약($$v\geq 0$$)까지 더하면 **플럭스 원추(flux cone)**라는 기하학적 구조가 만들어지며, 이 무한한 해 집합에서 하나의 최적해를 골라내는 방법(목적함수, 선형계획법)은 [Chapter 4](../chapter-4/README.md)에서, GPR·구획·바이오매스·교환 반응의 상세한 설계는 [Chapter 3](../chapter-3/README.md)에서 이어진다.

## 스스로 점검

1. **(개념)** 세포질의 피루브산(`pyr_c`)과 미토콘드리아의 피루브산(`pyr_m`)은 화학적으로 동일한 분자입니다. 그런데도 모델에서 서로 다른 대사물 노드로 취급하는 이유는 무엇일까요? *(힌트: 1.3절 — 대사물의 정체성은 화학종 자체가 아니라 그것이 관여하는 반응 집합으로 정의됩니다.)*

2. **(계산)** 반응 $$2\text{A} + \text{B} \rightarrow \text{C}$$ 하나만 있는 네트워크를 생각해 봅시다. 대사물 $$\{A, B, C\}$$를 행으로, 이 반응 하나를 열로 하는 $$3\times 1$$ 화학량론 행렬 $$\mathbf{S}$$를 직접 써 보세요. *(정답: $$\mathbf{S} = (-2, -1, +1)^\top$$)*

3. **(계산)** 4.3절 (B)의 열린 장난감 네트워크에서 $$v_1 = 4$$, $$v_3 = 2$$로 두면 $$v_0, v_2, v_4$$는 각각 얼마여야 $$\mathbf{S}\mathbf{v}=\mathbf{0}$$이 성립할까요? *(정답: $$v_0=6, v_2=4, v_4=6$$)*

4. **(개념)** 실제 게놈 규모 모델에서는 항상 $$n>m$$이 성립합니다. 이 부등식의 생물학적 의미는 무엇이며, 왜 이것이 곧바로 "무한히 많은 해"를 보장하지는 않는지(4.3절의 닫힌 네트워크 예시를 떠올리며) 설명해 보세요.

5. **(사고)** 만약 어떤 모델의 $$\mathbf{S}$$가 풀랭크(full row rank, 즉 $$r=m$$)라면, 이 모델에는 보존 화기가 존재할까요? 왜 그런지 4.5절의 왼쪽 영공간 정의를 이용해 설명해 보세요. *(힌트: 왼쪽 영공간의 차원은 $$m-r$$입니다.)*

6. **(그래프)** 3.2절의 핸드셰이크 관계에 따르면, 어떤 네트워크의 모든 대사물의 연결 정도(degree)를 더한 값과 모든 반응의 연결 정도를 더한 값은 항상 같습니다. *E. coli* core 모델의 비영 항목이 360개라면, 이 두 합은 각각 얼마일까요? *(정답: 두 값 모두 360입니다 — 같은 행렬의 비영 항목을 행 방향과 열 방향으로 각각 세어 더한 것이므로 항상 일치합니다.)*

7. **(계산)** 2.3절의 닫힌 장난감 네트워크에서 세 반응이 각각 $$v_1=2, v_2=2, v_3=5$$의 속도로 흐른다면(가상의 값), $$d\mathbf{x}/dt$$의 A, B, C 성분은 각각 얼마일까요? 이 값이 0이 아니라는 것은 무엇을 의미할까요? *(정답: A는 $$-1(2)+0(2)-1(5)=-7$$, B는 $$1(2)-1(2)+0(5)=0$$, C는 $$0(2)+1(2)+1(5)=7$$입니다. A가 순소비, C가 순생산되고 있다는 뜻이며, 이는 정상 상태가 아니라는 신호입니다 — 4.1절에서 다룬 것과 같은 상황입니다.)*

8. **(연결)** 4.5절에서 열린 장난감 네트워크의 영공간 기저 벡터 $$\mathbf{u}_1=(1,1,1,0,1)^\top$$과 $$\mathbf{u}_2=(1,0,0,1,1)^\top$$을 구했습니다. 이 두 벡터를 3.3절의 그래프 위 경로와 각각 연결해 설명해 보세요. *(정답: $$\mathbf{u}_1$$은 $$R_1,R_2$$가 1이고 $$R_3$$이 0이므로 "우회로" 경로 $$A\to R_1\to B\to R_2\to C$$에 대응하고, $$\mathbf{u}_2$$는 그 반대로 "직행로" 경로 $$A\to R_3\to C$$에 대응합니다. 행렬의 영공간 기저와 그래프의 독립 경로가 정확히 같은 대상을 가리키는 사례입니다.)*

다음 챕터들에서도 계속 재사용되는 핵심 기호를 표로 정리합니다. 이 표를 기억해 두면 이후 챕터의 수식을 훨씬 빠르게 읽을 수 있습니다.

| 기호 | 의미 | 처음 등장 |
|:---:|:---|:---|
| $$\mathbf{S}$$ | 화학량론 행렬 ($$m\times n$$) | 2.2절 |
| $$m, n$$ | 대사물 수, 반응 수 | 2.2절 |
| $$r = \operatorname{rank}(\mathbf{S})$$ | 계수(rank), 독립 방정식(또는 독립 열)의 개수 | 4.4절 |
| $$\mathbf{v}, v_j$$ | 통량 벡터, 개별 반응 $$j$$의 통량 (mmol·gDW$$^{-1}$$·h$$^{-1}$$) | 1.1절 |
| $$v_j^{lb}, v_j^{ub}$$ | 통량 하한/상한 | 1.2절 |
| $$\mathbf{x}, x_i$$ | 대사물 농도 벡터, 개별 대사물 $$i$$의 농도 | 4.1절 |
| $$n-r$$ | 정상 상태 통량 공간(영공간)의 진짜 차원(자유도) | 4.4절 |
| $$m-r$$ | 독립 보존 화기의 개수(왼쪽 영공간의 차원) | 4.5~4.6절 |

## 다음 장 예고

이 장에서 우리는 반응과 대사물이라는 두 개체를 화학량론 행렬 $$\mathbf{S}$$로 압축하고, $$\mathbf{S}\mathbf{v}=\mathbf{0}$$이라는 질량 보존 제약이 대사 네트워크에 어떤 자유도를 남기는지 손으로 직접 확인했습니다. 이 과정에서 우리는 같은 내용을 행렬(계산에 강함)과 그래프(직관에 강함)라는 두 언어로 오가는 법도 익혔습니다. 이제 우리는 네트워크의 "뼈대"를 갖췄습니다.

하지만 아직 빠진 것이 있습니다 — $$\mathbf{S}$$의 각 열(반응)이 **어떤 유전자·효소**에 의해 촉매되는지, 그 반응이 세포의 **어느 구획**에서 일어나는지, 그리고 세포와 환경의 **경계**는 어떻게 설정되는지에 대한 정보입니다. 이 정보들이 채워져야 비로소 $$\mathbf{S}$$가 "이름 없는 골격"에서 살아있는 세포의 완전한 모델로 거듭납니다. 다음 [Chapter 3. GEM의 구조](../chapter-3/README.md)에서는 GPR(Gene-Protein-Reaction) 규칙, 세포 구획과 수송 반응, 교환·생물량 반응의 설계를 다루며 이 골격에 생물학적 정체성을 입힙니다.

그다음에는 또 하나의 질문이 기다리고 있습니다 — 이 장에서 본 것처럼 $$\mathbf{S}\mathbf{v}=\mathbf{0}$$은 (심지어 비가역성까지 고려해도) 대개 무한히 많은 해, 즉 하나의 플럭스 원추를 허용합니다. 그렇다면 실제 세포는 이 무한한 가능성 중 "어느" 통량 분포를 선택하는 것일까요? [Chapter 4. Flux Balance Analysis (FBA)](../chapter-4/README.md)는 여기에 "생물학적으로 그럴듯한 목적(예: 최대 성장)을 최적화한다"는 가설로 답하며, 우리가 손으로 그려본 작은 부채꼴 모양의 가능 영역이 실제 iML1515 규모에서는 어떤 모습이 되는지, 그리고 그 위에서 최적해를 어떻게 계산하는지를 다룹니다.

## 핵심 용어 정리

| 용어 (English) | 정의 |
|:---|:---|
| 반응 (Reaction) | 대사물을 변환하는 기본 단위. 화학량론·방향성·통량 범위·GPR로 구성됨 |
| 대사물 (Metabolite) | 반응에서 소비·생성되는 화학 종. ID에 구획 접미사가 붙음 |
| 가역성 (Reversibility) | 반응이 양방향으로 진행 가능한지 여부. $$v^{lb}<0<v^{ub}$$로 코딩 |
| 구획 (Compartment) | 대사물·반응이 위치하는 세포 내 구역 (`_c`, `_e`, `_m` 등) |
| 화학량론 행렬 (Stoichiometric Matrix, S) | $$m \times n$$ 크기로 모든 반응의 화학량론 계수를 인코딩한 행렬 |
| 희소성 (Sparsity) | 행렬에서 0이 아닌 항목의 비율이 매우 낮은 성질. *E. coli* core 모델은 5.26% |
| 허브 대사물 (Hub Metabolite) | ATP, NADH, H$$_2$$O 등 수백 개 반응에 관여하는 소수의 고연결 대사물 |
| 이분 그래프 (Bipartite Graph) | 대사물 노드와 반응 노드 두 집합으로만 구성되고, 같은 집합 내 간선이 없는 그래프 |
| 정상 상태 가정 (Pseudo-Steady-State Assumption, PSSA) | 내부 대사물 농도가 관심 시간 척도에서 변하지 않는다는 가정, $$d\mathbf{x}/dt=\mathbf{0}$$ |
| 자유도 (Degrees of Freedom) | 정상 상태 제약만으로 유일해가 결정되지 않는 미지수의 여유분. 진짜 값은 $$n-r$$ ($$r$$=계수) |
| 계수 (Rank) | 행렬의 독립적인 행(또는 열)의 개수 $$r$$. 실제 GEM에서는 보통 $$r<m$$ |
| 영공간 (Null Space) | $$\{\mathbf{v}: \mathbf{S}\mathbf{v}=\mathbf{0}\}$$, 차원 $$n-r$$. 가능한 모든 정상 상태 통량 분포의 공간 |
| 왼쪽 영공간 (Left Null Space) | $$\{\mathbf{p}: \mathbf{p}^\top\mathbf{S}=\mathbf{0}\}$$, 차원 $$m-r$$. 보존 화기의 공간 |
| 열공간 (Column Space) | $$\{\mathbf{S}\mathbf{v}: \mathbf{v}\in\mathbb{R}^n\}$$, 차원 $$r$$. 네트워크가 달성 가능한 순생산/소비 속도 벡터의 공간 |
| 행공간 (Row Space) | $$\{\mathbf{S}^\top\mathbf{x}: \mathbf{x}\in\mathbb{R}^m\}$$, 차원 $$r$$. 질량 보존이 부과하는 통량 간 선형 의존성의 공간 |
| 보존 화기 (Conserved Moiety) | 왼쪽 영공간에서 유래하는, 총합이 항상 일정한 대사물 풀 (예: ATP+ADP+AMP). 교환 반응이 있으면 관련 화기의 보존이 깨질 수 있음 |
| 연결 정도 (Degree) | 이분 그래프에서 한 노드에 연결된 간선의 수. $$\mathbf{S}$$에서는 해당 행(또는 열)의 비영 항목 개수와 같음 |
| 핸드셰이크 관계 (Handshake Relation) | 모든 대사물의 연결 정도 합 = 모든 반응의 연결 정도 합 = 간선(비영 항목) 총개수 |
| 경로 (Path) | 이분 그래프에서 노드-간선을 번갈아 밟아가는 이동 순서열. 영공간의 기저 벡터 하나가 독립적인 경로 하나에 대응 |
| 회전 시간 (Turnover Time) | 대사물 농도를 그 순 소비/생산 속도로 나눈 시간 척도, $$\tau_i\approx x_i/\vert\sum_j S_{ij}v_j\vert$$. PSSA를 정당화하는 시간 척도 분리의 근거 |
| 플럭스 원추 (Flux Cone) | 영공간(등식 제약)과 비가역성의 부호 제약($$v\geq 0$$)을 함께 반영한 기하학적 해 집합. FBA 최적화가 이루어지는 무대 |
