# 5. 쌍대 문제: 그림자 가격, 환원비용, 강건성 분석, PhPP

## 5.1 LP의 쌍대성(Duality)

모든 LP에는 **쌍대 문제(Dual Problem)**가 존재하며, 쌍대 변수는 원(Primal) 문제 제약의 민감도 정보를 담고 있습니다. FBA의 원 문제에 대해 쌍대 문제는 세 가지 변수 집합을 도입합니다.

- $$\boldsymbol{\lambda} \in \mathbb{R}^m$$: $$m$$개의 정상상태 물질수지 제약 $$\mathbf{S}\mathbf{v}=\mathbf{0}$$에 대한 **그림자 가격**
- $$\boldsymbol{\alpha} \in \mathbb{R}^n_{\ge 0}$$: 하한 제약에 대한 쌍대 변수
- $$\boldsymbol{\beta} \in \mathbb{R}^n_{\ge 0}$$: 상한 제약에 대한 쌍대 변수

직관적으로, 쌍대 변수는 "이 제약을 아주 조금 풀어주면 최적값이 얼마나 좋아지는가?"에 대한 답입니다.

## 5.2 그림자 가격(Shadow Price) — 손으로 확인해보기

> **핵심 개념 · 용어(English):** **그림자 가격(Shadow Price)** $$\lambda_i$$ — 대사물 $$i$$의 물질수지 제약 우변($$b_i$$, 정상상태에서는 0)이 한 단위 완화될 때 최적 목적 함수값 $$Z^*$$가 얼마나 변하는지를 나타내는 민감도 지표: $$\lambda_i = \partial Z^*/\partial b_i$$.

여기서 중요한 구분이 하나 있습니다. **물질수지 등식의 그림자 가격**과 **영양분 섭취 bound의 민감도**는 같은 쌍대 변수가 아닙니다. 3절의 장난감 예제에서 포도당 공급 한계 $$v_1+v_2\le10$$을 11로 완화해 보겠습니다.

새 제약 하에서 다시 최적화하면: 여전히 효율 좋은 경로 2를 상한(8)까지 채우고, 남는 용량 $$11-8=3$$을 경로 1로 보냅니다($$v_1=3\le6$$이므로 가능). 새로운 최적값은:

$$Z^{*'} = 0.5(3)+0.8(8) = 1.5+6.4 = 7.9$$

$$\Delta Z^* = 7.9-7.4 = 0.5 \quad\text{(포도당 공급 1단위 증가당)}$$

$$0.5$$는 이 **섭취 용량 부등식**의 한계가치입니다. 반면 COBRApy의 `solution.shadow_prices`는 $$\mathbf{S}\mathbf{v}=\mathbf{b}$$라는 **대사물 물질수지 등식**의 쌍대값입니다. 내부 대사물의 $$b_i$$를 0에서 바꾼다는 것은 그 대사물에 미소한 source/sink를 허용하는 수학적 사고실험이며, 실제 배지에 그 대사물을 더 넣는 것과 곧바로 같지 않습니다. 영양 제한을 묻고 싶다면 해당 exchange reaction의 bound를 직접 조금 바꾸어 목적값의 유한차분을 계산하는 편이 명확합니다.

{% hint style="warning" %}
그림자 가격의 **부호는 목적함수 방향, 등식의 작성 방향, 솔버 관례**에 따라 달라집니다. 따라서 `음수 = 제한 영양소`처럼 부호만으로 보편적인 생물학적 분류를 만들면 안 됩니다. 또한 대안 최적해나 쌍대 퇴화가 있으면 쌍대값도 유일하지 않을 수 있고, 민감도는 현재 활성 제약 집합이 유지되는 작은 변화 범위에서만 국소적으로 유효합니다.
{% endhint %}

```python
solution = model.optimize()
shadow_prices = solution.shadow_prices

# 우선 절댓값이 큰 쌍대값을 확인하되, 부호를 곧바로 영양 제한으로 해석하지 않는다.
for met_id, sp in shadow_prices.abs().sort_values(ascending=False).head(10).items():
    name = model.metabolites.get_by_id(met_id).name
    signed_sp = shadow_prices[met_id]
    print(f"{met_id:15s} ({name:30s}): {signed_sp:10.6f}")

# 포도당 섭취 bound의 실제 국소 민감도는 유한차분으로 직접 확인한다.
base = model.slim_optimize()
with model:
    model.reactions.EX_glc__D_e.lower_bound -= 0.01
    perturbed = model.slim_optimize()
print("d(growth)/d(bound) ≈", (perturbed - base) / (-0.01))
```

## 5.3 환원비용(Reduced Cost) — 손으로 확인해보기

> **핵심 개념 · 용어(English):** **환원비용(Reduced Cost)** $$\bar{c}_j$$ — 최적해에서 플럭스가 상한이나 하한(경계)에 붙어 있는 반응 $$j$$에 대해, 그 경계 자체가 한 단위 완화될 때 목적 함수값이 얼마나 변하는지를 나타냄: $$\bar{c}_j = c_j - \mathbf{c}_B^\mathsf{T}\mathbf{B}^{-1}\mathbf{S}_{\cdot j}$$.

3.3절 최적해 $$(v_1^*, v_2^*)=(2,8)$$을 다시 보면, $$v_2$$는 자신의 상한 $$8$$에 딱 붙어 있습니다(경계에 위치, "비기저" 상태) 반면 $$v_1=2$$는 상·하한 어디에도 붙어 있지 않은 내부 값입니다("기저" 상태). 환원비용은 바로 이 "경계에 붙은" 변수에 대해서만 의미를 가집니다.

경로 2의 효소 용량 상한을 $$8$$에서 $$9$$로 한 단위 늘리면 어떻게 될까요? 여전히 포도당 공급 상한은 10이므로, 경로 2를 9까지 채우고 남는 $$10-9=1$$을 경로 1로 보냅니다.

$$Z^{*''} = 0.5(1)+0.8(9) = 0.5+7.2=7.7$$

$$\Delta Z^* = 7.7-7.4 = 0.3 \quad\text{(경로 2 용량 1단위 증가당)}$$

이 $$0.3$$이 바로 $$v_2$$의 **환원비용**입니다 — $$v_2$$가 이미 상한에 딱 붙어 "더 늘리고 싶지만 못 늘리는" 상태이기 때문에 0이 아닌 값을 가집니다. 반면 내부 값인 $$v_1$$의 환원비용은 정확히 0입니다(경계에 있지 않으므로 그 경계를 완화해도 아무 의미가 없습니다). 환원비용이 0에 가까운 반응은 "거의 활성화된" 반응으로, 실험적 조작이나 조절 개입의 후보가 됩니다.

```python
reduced_costs = solution.reduced_costs
significant = reduced_costs[reduced_costs.abs() > 1e-6].sort_values()
print(significant.head(10))
```

## 5.4 강건성 분석(Robustness Analysis)

**강건성 분석(Robustness Analysis)**은 특정 반응의 플럭스를 체계적으로 고정시켜 가며 목적 함수값(대개 성장률)이 어떻게 달라지는지 관찰하는 방법입니다(Edwards & Palsson, 2000). 수학적으로는 반응 $$j$$의 플럭스를 특정 값 $$\alpha$$로 고정하고 나머지에 대해 다시 최적화합니다.

$$Z(\alpha) = \max_{\mathbf{v}} \; \mathbf{c}^\mathsf{T}\mathbf{v} \quad \text{s.t.} \quad \mathbf{S}\mathbf{v}=\mathbf{0}, \; v_j=\alpha, \; \mathbf{v}^{lb}\le\mathbf{v}\le\mathbf{v}^{ub}$$

$$\alpha$$를 조금씩 바꿔가며 $$Z(\alpha)$$를 그리면 **강건성 곡선(Robustness Curve)**이 됩니다. 3절의 장난감 예제에서 $$v_2$$(경로 2 플럭스)를 0부터 8까지 고정해가며 최적 $$Z$$를 그려보면, $$Z$$가 $$v_2$$에 비례해 단조 증가하다가 $$v_2=8$$(상한)에서 꺾이는 곡선을 얻습니다 — 방금 5.3절에서 계산한 환원비용은 바로 이 곡선의 $$v_2=8$$ 지점에서의 기울기입니다.

| 곡선 형태 | 반응 유형 | 생물학적 의미 |
|:---|:---|:---|
| 단조 증가 | 유익 반응(Beneficial) | 플럭스가 늘수록 성장률 증가 |
| 단조 감소 → 0 | 필수 반응(Essential) | 플럭스가 줄면 성장률 감소, 완전 차단 시 사망 |
| 평평 → 0 | 중성 반응(Neutral) | 플럭스 변화가 성장에 영향 없음(대체 경로 존재) |

```python
import numpy as np

# 산소 흡수를 10으로 제한하고 포도당 흡수율을 0~20으로 스캔한다.
# 산소 제한을 명시하지 않으면 textbook의 기본 산소 하한은 -1000이라
# 이 범위에서 성장률이 거의 선형으로 증가한다.
glc_range = np.linspace(0, 20, 41)
growth = []
for glc in glc_range:
    with model:
        model.reactions.EX_o2_e.lower_bound = -10
        model.reactions.EX_glc__D_e.lower_bound = -glc
        value = model.slim_optimize(error_value=np.nan)
        growth.append(value if np.isfinite(value) else 0.0)
# 낮은 포도당 구간에서는 빠르게 증가하고, 호흡 용량이 제한되면 발효를
# 함께 사용하면서 증가 기울기가 작아진다. "평평한 plateau"라고 단정하지 않는다.
```

*그림 4.2: COBRApy 0.30.0 `textbook` 모델의 실제 계산 결과입니다. 파란 선은 산소가 사실상 무제한인 조건, 주황 선은 산소 uptake cap을 10으로 둔 조건입니다. 같은 포도당 bound라도 다른 활성 제약이 바뀌면 곡선의 기울기와 꺾임이 달라지므로, 한 반응만 보고 “포화”를 일반화하면 안 됩니다.*

## 5.5 표현형 상 평면 분석(Phenotypic Phase Plane, PhPP)

강건성 분석을 반응 하나가 아니라 **두 반응**(보통 포도당·산소 섭취율)으로 확장하면 **표현형 상 평면(PhPP)**이 됩니다. 그림자 가격은 PhPP에서 정성적으로 다른 대사 전략의 영역을 구분하는 데 사용됩니다. PhPP의 각 영역 내부에서는 그림자 가격이 일정하며, 영역 경계를 지날 때 불연속적으로 변화합니다 — 이는 LP 최적해의 기저(Basis)가 바뀌는, 즉 **대사 전략이 급격히 전환**되는 지점입니다(예: 완전 호흡 → 아세테이트 분비를 동반한 발효).

```
      O₂ 흡수율
        ^
        │    [Region I: 산소 제한]
        │    (완전 호흡, λ_O2 > 0)
        │         ╲
        │          ╲  [전이 영역: 아세테이트 분비]
        │           ╲
        │            ╲ [Region II: 포도당 제한]
        │             ╲(λ_Glc > 0, 발효 대사)
        └──────────────────> Glucose 흡수율
```

이러한 영역 전환은 효모의 **Crabtree 효과**(산소가 충분해도 고농도 포도당에서 발효를 선호)나 암세포의 **Warburg 효과**(호기적 해당과정 선호)와 같은 현상을 해석하는 한 가지 제약 기반 틀을 제공합니다. 그림자 가격은 현재 최적 기저가 유지되는 범위에서만 국소적으로 유효합니다. COBRApy의 일반 `Solution` 객체가 그 허용 범위를 자동 보고하는 것은 아니므로, 실제 강건성은 bound를 스캔하고 다시 최적화해 확인해야 합니다. PhPP를 균주 설계(strain design)의 production envelope와 결합하는 응용은 [Chapter 8](../chapter-8/README.md)에서 더 다룹니다.

## 5.6 쌍대 LP를 처음부터 만들어보기 — 라그랑주에서 손 계산까지

지금까지는 그림자 가격과 환원비용을 "제약을 조금 완화하고 다시 풀어서" 수치로 확인했습니다(5.2, 5.3절). 이번에는 거꾸로, **쌍대 문제(Dual Problem)를 원 문제로부터 직접 만들어** 같은 숫자가 이론적으로도 나온다는 것을 확인해 봅시다.

**표준 LP 쌍대 정리.** 다음과 같은 형태의 원(Primal) LP를 생각합니다.

$$
\begin{aligned}
\text{(원 문제)}\quad\max_{\mathbf{v}} \quad & \mathbf{c}^\mathsf{T}\mathbf{v} \\
\text{s.t.} \quad & \mathbf{A}\mathbf{v} \le \mathbf{b} \\
& \mathbf{v} \ge \mathbf{0}
\end{aligned}
$$

이 문제의 **쌍대 문제**는 다음과 같이 정의됩니다 — 원 문제의 제약 개수(행)만큼 쌍대 변수 $$\mathbf{y}$$를 만들고, 원 문제의 변수 개수(열)만큼 쌍대 제약을 만드는 "행과 열을 뒤바꾸는" 대응 관계입니다.

$$
\begin{aligned}
\text{(쌍대 문제)}\quad\min_{\mathbf{y}} \quad & \mathbf{b}^\mathsf{T}\mathbf{y} \\
\text{s.t.} \quad & \mathbf{A}^\mathsf{T}\mathbf{y} \ge \mathbf{c} \\
& \mathbf{y} \ge \mathbf{0}
\end{aligned}
$$

**상보 여유성(Complementary Slackness)의 두 규칙:**

1. 원 문제의 제약 $$i$$가 빠듯하지 않으면($$(\mathbf{A}\mathbf{v})_i < b_i$$), 그 쌍대 변수는 $$y_i=0$$이어야 합니다.
2. 원 문제의 변수 $$v_j$$가 0이 아니면(경계에 있지 않으면), 그 쌍대 제약은 등식으로 빠듯하게 걸립니다: $$(\mathbf{A}^\mathsf{T}\mathbf{y})_j = c_j$$.

**토이 예제에 그대로 적용하기.** 3.1절의 LP는 $$\mathbf{c}=(0.5,\,0.8)^\mathsf{T}$$, 그리고

$$
\mathbf{A}=\begin{pmatrix}1&1\\1&0\\0&1\end{pmatrix},\qquad
\mathbf{b}=\begin{pmatrix}10\\6\\8\end{pmatrix}
$$

(세 행은 각각 $$v_1+v_2\le10$$, $$v_1\le6$$, $$v_2\le8$$입니다.) 쌍대 변수를 각 제약에 대응해 $$y_1,y_2,y_3\ge0$$이라 하면, 쌍대 문제는

$$
\min\ 10y_1+6y_2+8y_3 \quad\text{s.t.}\quad y_1+y_2\ge0.5,\ \ y_1+y_3\ge0.8,\ \ \mathbf{y}\ge\mathbf{0}
$$

이제 3.3절에서 구한 원 문제의 최적해 $$(v_1^*,v_2^*)=(2,8)$$을 상보 여유성 규칙에 대입합니다.

- 제약 2($$v_1\le6$$)는 $$v_1^*=2<6$$이므로 빠듯하지 않습니다 $$\Rightarrow y_2=0$$.
- 제약 1($$v_1+v_2\le10$$)은 $$2+8=10$$으로 빠듯합니다 $$\Rightarrow y_1$$은 0이 아닐 수 있습니다.
- 제약 3($$v_2\le8$$)은 $$v_2^*=8$$로 빠듯합니다 $$\Rightarrow y_3$$은 0이 아닐 수 있습니다.
- $$v_1^*=2\neq0$$이므로 첫 번째 쌍대 제약은 등식: $$y_1+y_2=0.5$$.
- $$v_2^*=8\neq0$$이므로 두 번째 쌍대 제약도 등식: $$y_1+y_3=0.8$$.

$$y_2=0$$을 첫 식에 대입하면 $$y_1=0.5$$, 이를 둘째 식에 대입하면 $$y_3=0.8-0.5=0.3$$입니다.

$$\boxed{y_1=0.5,\quad y_2=0,\quad y_3=0.3}$$

이 세 숫자를 다시 살펴봅시다. $$y_1=0.5$$는 5.2절에서 포도당 공급 상한을 $$10\to11$$로 완화했을 때 손으로 계산한 $$\Delta Z^*=0.5$$와 정확히 같고, $$y_3=0.3$$은 5.3절에서 경로 2의 상한을 $$8\to9$$로 늘렸을 때 확인한 환원비용 $$0.3$$과 정확히 같으며, $$y_2=0$$은 빠듯하지 않은 제약(경로 1의 상한)의 쌍대 값이 0이라는 원리와 정확히 일치합니다. 또한 이 값들은 4.4절 심플렉스 최종 타블로의 $$Z=7.4-0.5s_1-0.3s_3$$에서 읽은 계수와도 부호까지 정확히 일치합니다 — **손 계산, 심플렉스 타블로, 쌍대 LP라는 세 개의 서로 다른 경로가 정확히 같은 결론에 도달**한 것입니다.

{% hint style="warning" %}
이 절에서는 부등식이 모두 "$$\le$$, $$v\ge0$$" 형태인 표준 쌍대 정리를 정확히 그대로 적용했기 때문에 부호가 깔끔하게 나왔습니다. 실제 genome-scale FBA는 등식 제약 $$\mathbf{S}\mathbf{v}=\mathbf{0}$$과 양방향 경계 $$\mathbf{v}^{lb}\le\mathbf{v}\le\mathbf{v}^{ub}$$를 함께 가지므로, 5.1절의 $$\boldsymbol\lambda,\boldsymbol\alpha,\boldsymbol\beta$$는 이 토이 예제의 $$\mathbf{y}$$를 등식·양방향 부등식까지 확장한 대응물입니다. 등식을 어떻게 쓰느냐(예: $$\mathbf{S}\mathbf{v}=\mathbf{0}$$ 대 $$-\mathbf{S}\mathbf{v}=\mathbf{0}$$)에 따라 $$\boldsymbol\lambda$$의 부호가 바뀔 수 있다는 것이 5.2절 첫 경고의 정확한 이유입니다. 부호 규약과 무관하게, **상보 여유성이라는 논리(빠듯한 제약만 0이 아닌 쌍대 값을 가진다)는 항상 성립**합니다.
{% endhint %}

## 5.7 환원비용 공식을 기저 역행렬로 직접 검증하기

5.3절 정의 상자에서 환원비용을 $$\bar{c}_j = c_j - \mathbf{c}_B^\mathsf{T}\mathbf{B}^{-1}\mathbf{S}_{\cdot j}$$이라는 행렬식으로 소개했지만, 그때는 숫자를 직접 대입해 보지 않았습니다. 이번에는 4.4절 심플렉스 타블로의 최종 기저를 그대로 가져와 이 공식이 정말로 같은 답을 주는지 확인해 봅시다.

**기저 행렬 구성.** 4.4절의 마지막 타블로에서 기저 변수는 $$\{v_1, v_2, s_2\}$$였습니다(비기저는 $$s_1, s_3$$). 세 제약식의 계수를 이 세 변수 순서로 뽑아 기저 행렬 $$\mathbf{B}$$를 만듭니다.

$$
\mathbf{B} = \begin{pmatrix} 1&1&0 \\ 1&0&1 \\ 0&1&0 \end{pmatrix}
\qquad(\text{열 순서: } v_1,\ v_2,\ s_2)
$$

가우스 소거나 여인수 전개로 역행렬을 구하면:

$$
\mathbf{B}^{-1} = \begin{pmatrix} 1&0&-1 \\ 0&0&1 \\ -1&1&1 \end{pmatrix}
$$

**쌍대 변수를 행렬 공식으로 재계산.** 기저 변수의 목적계수 $$\mathbf{c}_B=(c_{v_1},c_{v_2},c_{s_2})^\mathsf{T}=(0.5,\ 0.8,\ 0)^\mathsf{T}$$를 이용해 $$\mathbf{y}^\mathsf{T}=\mathbf{c}_B^\mathsf{T}\mathbf{B}^{-1}$$을 계산합니다.

$$
\mathbf{c}_B^\mathsf{T}\mathbf{B}^{-1} = 0.5\,(1,0,-1) + 0.8\,(0,0,1) + 0\,(-1,1,1) = (0.5,\ 0,\ 0.3)
$$

$$(0.5,\ 0,\ 0.3)$$ — 이것이 바로 5.6절에서 상보 여유성으로 손 계산했던 $$(y_1,y_2,y_3)$$와 정확히 같습니다! 이번에는 "제약을 완화하고 다시 풀어서" 얻은 것이 아니라, **기저 역행렬이라는 순수한 선형대수 공식**만으로 같은 답에 도달했습니다.

**환원비용도 같은 방식으로.** 비기저 변수 $$s_1$$과 $$s_3$$의 열벡터는 각각 $$\mathbf{S}_{\cdot s_1}=(1,0,0)^\mathsf{T}$$, $$\mathbf{S}_{\cdot s_3}=(0,0,1)^\mathsf{T}$$이고, 둘 다 목적계수 $$c_{s_1}=c_{s_3}=0$$입니다.

$$
\bar{c}_{s_1} = c_{s_1} - \mathbf{c}_B^\mathsf{T}\mathbf{B}^{-1}\mathbf{S}_{\cdot s_1} = 0 - (0.5,0,0.3)\cdot(1,0,0)^\mathsf{T} = -0.5
$$

$$
\bar{c}_{s_3} = c_{s_3} - \mathbf{c}_B^\mathsf{T}\mathbf{B}^{-1}\mathbf{S}_{\cdot s_3} = 0 - (0.5,0,0.3)\cdot(0,0,1)^\mathsf{T} = -0.3
$$

이 값 $$-0.5$$와 $$-0.3$$은 4.4절 최종 타블로 $$Z=7.4-0.5s_1-0.3s_3$$의 계수와 부호까지 정확히 같습니다. 이제 손 계산(5.2~5.3절), 심플렉스 타블로(4.4절), 쌍대 LP의 상보 여유성(5.6절), 기저 역행렬 공식(이 절)이라는 **네 가지 서로 다른 방법이 모두 같은 결론**에 도달했다는 것을 확인했습니다. COBRApy가 `solution.shadow_prices`, `solution.reduced_costs`를 계산할 때 내부적으로 수행하는 것이 바로 이 마지막 방법 — 최적 기저의 역행렬을 이용한 선형대수 계산 — 입니다.

{% hint style="info" %}
💡 **팁:** 손으로 $$3\times3$$ 역행렬을 구하는 것이 부담스럽다면, `numpy.linalg.inv`로 즉시 검산할 수 있습니다.
```python
import numpy as np
B = np.array([[1,1,0],[1,0,1],[0,1,0]])
B_inv = np.linalg.inv(B)
c_B = np.array([0.5, 0.8, 0.0])
y = c_B @ B_inv
print(y)   # 기대 출력: [0.5  0.   0.3]
```
{% endhint %}
