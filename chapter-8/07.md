# 7. Production Envelope와 Phenotype Phase Plane

## 7.1 Production Envelope: 생장-생산 트레이드오프의 시각화

**동기.** 균주를 설계했다면, "이 설계가 정말 growth-coupled인가?"를 한눈에 확인하고 싶습니다. 그 그림이 바로 production envelope입니다.

**Production envelope(생산 포락선)**는 생장율(가로축)과 목표 산물의 분비 flux(세로축) 사이의 실현 가능한 관계를 그린 그래프입니다. 각 생장율 값에서 [FVA](../chapter-4/README.md)를 수행해 그 생장율을 유지하는 조건에서 산물 flux의 최댓값·최솟값을 구하면, 포락선의 상한선과 하한선이 얻어집니다.

**손으로 꼭짓점을 읽어 봅시다.** 어떤 균주의 포락선이 다음 꼭짓점을 가진다고 합시다(모식적 수치).

- 점 A = (생장율 0, 산물 flux 20): 세포가 전혀 자라지 않으면 모든 탄소를 산물로 돌려 최대 20까지 생산 가능.
- 점 B = (생장율 0.8, 산물 flux 0): 산물을 전혀 안 만들고 최대 성장(0.8)만 추구 가능 — **non-coupled**.
- 점 B' = (생장율 0.8, 산물 flux 5): 최대 성장 지점에서도 산물 하한이 5 — **growth-coupled**.

핵심은 **최대 생장율 지점에서 산물 flux의 하한**을 읽는 것입니다.

| 포락선 형태 | 최대 생장율에서 산물 하한 | 해석 |
|:---|:---:|:---|
| B'처럼 하한 > 0 | 예 (예: 5) | **Growth-coupled** — 자라려면 반드시 생산 (OptKnock이 지향) |
| B처럼 하한 = 0 | 아니오 (0) | Non-coupled — 생산 없이도 최적 생장, 진화 중 생산성 소실 위험 |
| 상·하한 폭이 넓음 | — | 같은 생장률에서 산물 flux의 대안해 범위가 큼 (§6.6 RobustKnock이 다루는 문제) |

**모식적 수치가 아니라 실제로 꼭짓점을 유도해 봅시다 — §6.2의 장난감 네트워크를 그대로 사용합니다.** 흡수 $$v_1$$($$0\le v_1\le10$$), 효율적 성장 경로 $$v_2$$($$A\to$$바이오매스, 계수 1), 생산 결합 경로 $$v_3$$($$A\to$$바이오매스+산물, 각각 계수 0.5)로 이루어진 네트워크에서, **결손 없는(wild-type) 모델**의 envelope를 생장률 $$\mu$$의 함수로 직접 구해 봅시다.

바이오매스 $$\mu = v_2+0.5v_3$$을 고정한 채 $$v_3$$(따라서 산물 $$p=0.5v_3$$)가 가질 수 있는 범위를 구합니다. $$v_2=\mu-0.5v_3 \ge 0$$에서 $$v_3 \le 2\mu$$, 그리고 $$v_1=v_2+v_3=\mu+0.5v_3 \le 10$$에서 $$v_3 \le 20-2\mu$$를 얻습니다. $$v_3\ge0$$이 항상 가능하므로(즉 $$v_2=\mu$$로 전부 채우면 됨), 결국

$$
0 \;\le\; v_3 \;\le\; \min(2\mu,\; 20-2\mu) \quad\Longrightarrow\quad 0 \;\le\; p=0.5v_3 \;\le\; \min(\mu,\; 10-\mu)
$$

입니다. 즉 이 wild-type 네트워크의 production envelope는 모든 $$\mu$$에서 **하한이 항상 0**이고, 상한은 $$\mu\le5$$에서 $$p_{\max}=\mu$$(원점을 지나는 기울기 1 직선), $$\mu>5$$에서 $$p_{\max}=10-\mu$$(내려가는 직선)인 **삼각형(텐트) 모양**입니다. 세 꼭짓점은 $$(\mu,p)=(0,0)$$, $$(10,0)$$(최대 생장점, 상·하한 모두 0), 그리고 정점 $$(5,5)$$입니다. 최대 생장점 $$(10,0)$$에서 하한이 0인 것은 물론 상한마저 0이라는 뜻으로, "최대로 자라려면 산물을 전혀 만들 수 없다"는 강한 non-coupling을 보여줍니다.

이제 §6.2에서 $$v_2$$를 knockout한 **OptKnock 뮤턴트**의 envelope를 구해 봅시다. $$v_2=0$$이 강제되므로 $$\mu=0.5v_3$$, $$p=0.5v_3$$이 되어 $$p=\mu$$가 예외 없이 그대로 성립합니다($$v_1=v_3\le10$$이므로 $$\mu\in[0,5]$$). 상한도 하한도 같은 직선 $$p=\mu$$ 위에 있는, 폭이 0인 "선분형" envelope입니다 — 이것이 완벽한 growth-coupling의 극단적인 예이며, 그 끝점 $$(5,5)$$는 §6.2에서 bilevel MILP로 구한 OptKnock 최적해와 정확히 일치합니다. 요컨대 production envelope는 OptKnock이 찾아낸 "하나의 점"을 생장률 전 구간으로 확장해 보여주는 그림입니다.

| 모델 | envelope 모양 | 최대 생장점 $$(\mu,p)$$ | 해석 |
|:---|:---|:---:|:---|
| Wild-type (v2 있음) | 삼각형(텐트), 정점 (5,5) | $$(10,0)$$ | 하한·상한 모두 0 — 강한 non-coupled |
| $$v_2$$ knockout | 직선 $$p=\mu$$ | $$(5,5)$$ | 하한=상한=$$\mu$$ — 완벽한 growth-coupled |

## 7.2 Phenotype Phase Plane (PhPP)로의 확장

**아주 쉬운 도입.** production envelope가 "생장률 하나를 고정했을 때 산물이 얼마나 나오는가"를 보여줬다면, PhPP는 한 걸음 더 나아가 "환경 조건(예: 산소와 포도당의 공급 비율) 자체가 바뀌면 세포의 최적 전략이 어떻게 달라지는가"를 지도로 그립니다. 마치 등산로 지도에서 산소 공급을 위도, 포도당 공급을 경도로 삼아 "이 좌표에서는 어떤 등반 전략(대사 경로 조합)이 최선인가"를 색칠해 보여주는 것과 비슷합니다.

**PhPP (Phenotype Phase Plane)**는 두 환경 변수(전형적으로 두 기질의 흡수율)를 축으로 하여 각 조합에서의 최적 대사 전략을 지도화한 것입니다. [Chapter 4](../chapter-4/README.md)의 로버스트니스 분석·shadow price 개념을 2차원으로 확장한 것으로, 평면은 **Line of Optimality (LO)** — 두 기질이 최적 비율로 소비되는 경계선 — 에 의해 영역이 나뉘며, 각 영역 내에서는 shadow price가 일정하게 유지되다가 영역 경계를 넘으면 불연속적으로 바뀝니다.

**직관을 숫자로 붙잡아 봅시다.** 포도당과 산소를 두 축으로 하는 PhPP를 생각해 보십시오. 산소가 충분히 공급되는 영역에서는 세포가 호흡(산화적 인산화)으로 ATP를 효율적으로 뽑아내는 전략이 최적이며, 이 영역에서는 "포도당을 한 단위 더 주면 생장률이 얼마나 느는가"를 나타내는 shadow price가 어떤 일정한 값(예: 단위 포도당당 생장률 증가 0.5)으로 유지됩니다. 그런데 산소 공급이 부족해 호흡 전략을 쓸 수 없는 영역으로 넘어가면, 세포는 대신 발효(포도당 1단위당 ATP 수율이 더 낮은 경로)로 전략을 바꾸며, shadow price도 더 낮은 값(예: 0.2)으로 **불연속적으로** 뚝 떨어집니다. 이 두 영역을 가르는 경계선이 바로 Line of Optimality이며, 경계를 넘을 때마다 "어느 반응의 조합이 활성화되는가"라는 최적 basis 자체가 바뀝니다 — [Chapter 4](../chapter-4/README.md)에서 다룬 FBA의 대안 최적해·축퇴(degeneracy) 개념과 정확히 같은 현상이 여기서는 2차원 환경 공간에 펼쳐진 것입니다.

균주 설계 맥락에서는 두 축 중 하나를 **산물 분비율**로 대체한 3차원 PhPP로, 기질·산소 조건에 따라 production envelope가 어떻게 이동하는지 탐색할 수 있습니다. 예컨대 §7.1의 삼각형 envelope가 산소 조건에 따라 정점의 높이(최대 산물량)와 위치(그 산물을 내는 생장률)가 이동하는 모습을 관찰할 수 있습니다. PhPP의 일반 이론(shadow price, 로버스트니스 분석)은 [Chapter 4](../chapter-4/README.md)를 참고하십시오.

## 7.3 배지 조건 최적화와 Pareto Frontier

산소·탄소원 흡수율 같은 배지 조건은 production envelope의 형태 자체를 바꿉니다 — 혐기 조건은 발효 부산물(에탄올, 아세테이트, 젖산) 생산을 촉진하지만 ATP 수율은 낮추므로, 목표 산물에 따라 최적 산소 조건이 다릅니다. 생장율과 산물 수율은 흔히 상충하는 두 목적이므로, 여러 균주 설계 후보를 **Pareto frontier**(한 목적을 희생하지 않고는 다른 목적을 개선할 수 없는 경계) 위에 배치해 비교하는 것이 실무적으로 유용합니다.

**손으로 Pareto frontier를 가려내 봅시다.** 다섯 개의 균주 설계 후보가 있고, 각각의 (생장률, 산물 수율)을 측정했다고 합시다.

| 후보 | 생장률 ($$\text{h}^{-1}$$) | 산물 수율 (mol/mol) |
|:---:|:---:|:---:|
| 설계 1 | 0.8 | 0.1 |
| 설계 2 | 0.6 | 0.5 |
| 설계 3 | 0.4 | 0.6 |
| 설계 4 | 0.3 | 0.4 |
| 설계 5 | 0.1 | 0.7 |

설계 4는 설계 3과 비교하면 생장률(0.3<0.4)도 낮고 수율(0.4<0.6)도 낮습니다 — 두 목적 모두에서 설계 3에게 **지배(dominate)당하므로** 설계 4는 Pareto frontier에서 탈락합니다("어느 목적도 희생하지 않고 개선할 수 있는" 대안이 이미 있기 때문입니다). 나머지 네 후보(1, 2, 3, 5)는 서로를 지배하지 않습니다 — 예컨대 설계 2는 설계 3보다 생장률은 높지만(0.6>0.4) 수율은 낮으므로(0.5<0.6), 어느 쪽이 "더 낫다"고 한마디로 말할 수 없는 진짜 트레이드오프 관계입니다. 따라서 이 다섯 후보 중 Pareto frontier는 **{설계 1, 설계 2, 설계 3, 설계 5}**이며, 최종 선택은 "생장 속도가 더 중요한가, 수율이 더 중요한가"라는 공정·경제성 판단(생산 비용, 정제 비용, 배양 시간)에 달려 있습니다 — 이는 순수하게 계산만으로는 답할 수 없는 부분입니다.

> 💡 **실습: 배지 조건 스캔과 Pareto 비교**

```python
# 목적: 환경별 최대 생장의 일정 비율을 유지하면서 목표 산물을 최대화한다.
import numpy as np
import pandas as pd

def medium_scan(model, target_rxn_id, o2_range, glc_range, growth_fraction=0.9):
    rows = []
    for o2 in o2_range:
        for glc in glc_range:
            with model as m:
                m.reactions.EX_o2_e.lower_bound = -o2
                m.reactions.EX_glc__D_e.lower_bound = -glc
                max_growth = m.slim_optimize(error_value=np.nan)
                # 탄소원이 0인 조건처럼 infeasible이면 NaN을 bound에 넣지 않는다.
                if not np.isfinite(max_growth) or max_growth <= 0:
                    rows.append({"o2": o2, "glc": glc,
                                 "growth": np.nan, "production": np.nan,
                                 "unconstrained_max_growth": np.nan,
                                 "status": "infeasible"})
                    continue
                required_growth = growth_fraction * max_growth
                m.reactions.Biomass_Ecoli_core.lower_bound = required_growth
                m.objective = target_rxn_id
                product_solution = m.optimize()
                if product_solution.status == "optimal":
                    growth = product_solution.fluxes["Biomass_Ecoli_core"]
                    production = product_solution.objective_value
                else:
                    growth = production = np.nan
            rows.append({"o2": o2, "glc": glc, "growth": growth,
                         "production": production,
                         "unconstrained_max_growth": max_growth,
                         "status": product_solution.status})
    return pd.DataFrame(rows)

def pareto_frontier(points):
    """(growth, production) 쌍 중 어느 것에도 지배되지 않는 점만 남긴다"""
    pts = sorted(points, key=lambda p: (-p[0], -p[1]))
    frontier, best_prod = [], -np.inf
    for growth, prod in pts:
        if prod > best_prod:
            frontier.append((growth, prod))
            best_prod = prod
    return frontier
```
