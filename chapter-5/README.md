# Chapter 5. 모델 재구축과 품질 관리

게놈 규모 대사 모델은 게놈 서열을 반응 목록으로 그대로 옮긴 결과가 아니다. 세포 안에서 일어나는 대사를 컴퓨터로 다루려면 여러 지식을 하나로 엮어야 한다. 유전자 기능 주석, 반응 화학량론, 세포 구획, 반응 방향성, 수송, 바이오매스 조성, 실험 표현형을 추적 가능한 하나의 지식 표현으로 통합한다. 그런 다음 계산이 되는지와 생물학적으로 타당한지를 반복 검증한다. 그래서 **[재구축(reconstruction)](../glossary.md)**과 **모델(model)**을 구분한다. 재구축은 지식을 조직하는 과정이고, 모델은 그 재구축에 경계 조건과 목적함수를 부여한 계산 객체다.

이 장은 미생물 GEM의 초안 작성에서 출발해 수동 큐레이션, gap-filling, 표준 형식, 자동 시험 및 독립 표현형 검증까지 이어지는 전 과정을 다룬다. 후반부에서는 같은 원리가 인체 범용 모델의 통합과 조직 특이 모델 추출에서 어떻게 확장되는지 검토한다. 자동화 도구의 출력은 재구축의 출발점일 수 있지만, 그 자체가 검증된 생물학적 결론은 아니다.

## 표기와 읽기 원칙

이 책은 한국어 용어를 먼저 쓰고, 처음 등장할 때 영어 원어와 약어를 함께 표기한다. 이후에는 같은 장 안에서 한 표기를 일관되게 사용한다.

- **[플럭스](../glossary.md)**(flux; 대사 통량), **반응**(reaction), **대사물**(metabolite)은 각각 단위 시간당 반응 진행률, 화학량론적 변환, 구획을 포함한 화학종을 뜻한다.
- **경계조건**(bounds), **목적함수**(objective function), **솔버**(solver)는 생물학적 사실이 아니라 모델에 부여한 계산 조건이다.
- 조건·가정·절차는 번호 목록으로, 결과의 범위와 예외는 `해석상의 주의` 상자로 구분한다.

{% hint style="warning" %}
본문의 “예측”은 명시된 모델·배지·경계조건·목적함수 아래의 계산 결과다. 실험 관찰이나 인과적 효과와 혼용하지 않는다.
{% endhint %}

## 재구축의 증거 흐름

```mermaid
flowchart LR
    G["게놈·단백질 서열"] --> A["기능 주석과 상동성 검색"]
    D["반응 DB·참조 모델"] --> A
    A --> R["초안 재구축"]
    R --> C["문헌 기반 수동 큐레이션"]
    C --> M["계산 모델<br/>S, bounds, objective"]
    M --> Q["구조·일관성·회귀 시험"]
    Q --> P["조건별 표현형 검증"]
    P -->|불일치| C
    P --> V["버전이 명시된 배포 모델"]
```

*Figure 5.1: GEM 재구축의 증거 흐름. 자동 시험과 표현형 검증은 서로 대체할 수 없으며, 불일치는 근거가 기록된 큐레이션 단계로 환류한다. 저자 작성; 절차 구성은 [Thiele & Palsson (2010)](https://doi.org/10.1038/nprot.2009.203)과 [Lieven et al. (2020)](https://doi.org/10.1038/s41587-020-0446-y)을 바탕으로 재구성.*

## 이 장을 읽는 방법

재구축은 자동 도구가 한 번 실행되면 끝나는 작업이 아니라, **근거를 기록하고 오류를 찾으며 검증하는 반복 과정**이다. 각 반응을 읽을 때 “무엇을 근거로 넣었는가?”와 “어떤 조건에서 검증했는가?”를 함께 묻는다.

1. 유전자 주석과 상동성은 반응 후보를 제안한다.
2. 수동 정제는 화학량론·구획·GPR·출처 이력을 검토한다.
3. 갭 필링은 누락 반응의 가설을 만들며, 독립 자료로 검증해야 한다.
4. 품질 관리는 파일 형식 검사와 생물학적 과제 검증을 모두 포함한다.

{% hint style="warning" %}
학습에 쓴 성장·분비·과제를 같은 모델의 성능 평가에도 쓰면 성능이 과대평가될 수 있다. hold-out 조건을 별도로 둔다.
{% endhint %}

## 학습 목표

이 장을 마치면 다음을 수행할 수 있다.

1. 서열 정렬의 E-value, identity 및 coverage를 구분하고, 상동성 근거가 반응 할당으로 변환되는 과정의 불확실성을 설명한다.
2. Thiele–Palsson 프로토콜의 네 단계를 구분하고 반응별 근거, 방향성, GPR 및 구획 정보를 추적 가능하게 기록한다.
3. CarveMe, ModelSEED, gapseq 및 수동 재구축의 입력·가정·출력을 비교한다.
4. [gap-filling](../glossary.md)을 [혼합정수선형계획](../glossary.md) 문제로 정식화하고, 최소 반응 수와 생물학적 근거 사이의 차이를 평가한다.
5. [질량·전하 균형](../glossary.md), [화학량론적 일관성](../glossary.md), flux consistency, 주석 완전성 및 표현형 정확도를 서로 구분한다.
6. [SBML FBC](../glossary.md)와 Groups package의 역할을 구분하고, 버전·solver·배지·bounds가 고정된 회귀 검증 계획을 작성한다.
7. 인체 범용 모델과 조직 특이 모델의 구축 목적을 구분하고 tINIT의 metabolic-task 보존 원리를 설명한다.
