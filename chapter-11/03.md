# 3. pFBA와 FVA

## 3.1 Parsimonious FBA(pFBA)

FBA의 최적 성장률을 유지하는 flux 분포는 하나가 아닙니다. pFBA는 최적 성장을 유지한 채 총 flux $$\sum_j|v_j|$$를 최소화해, 같은 성장률을 내는 해 중 “불필요한 우회가 없는” 대표 해를 고릅니다([Chapter 4](../chapter-4/README.md)).

```python
from cobra.flux_analysis import pfba

p = pfba(model)
print("pFBA growth:", round(p.fluxes["Biomass_Ecoli_core"], 6))
print("sum |v| FBA :", round(sol.fluxes.abs().sum(), 2))
print("sum |v| pFBA:", round(p.fluxes.abs().sum(), 2))
```

```
pFBA growth: 0.873922
sum |v| FBA : 518.42
sum |v| pFBA: 518.42
```

이 모델·solver 조합에서는 GLPK가 반환한 기본 FBA 해의 총 flux가 이미 pFBA 해와 같습니다(518.42). 이는 **항상 성립하는 성질이 아니라** 이 모델·solver·조건에서 관찰된 결과입니다. solver나 모델이 바뀌면 FBA 기본 해가 불필요한 순환을 포함해 총 flux가 더 클 수 있으며, 그때 pFBA와 값이 갈립니다([Chapter 4](../chapter-4/README.md) 8절).

## 3.2 Flux Variability Analysis(FVA)

FVA는 최적 성장의 일정 비율 이상을 유지한다는 조건에서 각 반응을 따로 최소·최대화해 flux가 취할 수 있는 범위를 구합니다. 범위의 폭이 0보다 크면 대안 최적해가 존재한다는 뜻입니다.

```python
from cobra.flux_analysis import flux_variability_analysis

fva = flux_variability_analysis(
    model,
    reaction_list=["PGI", "PFK", "PYK", "G6PDH2r", "EX_ac_e", "EX_etoh_e", "EX_for_e"],
    fraction_of_optimum=0.90, processes=1,
)
fva["width"] = fva["maximum"] - fva["minimum"]
print(fva.round(4).to_string())
```

```
           minimum  maximum    width
PGI       -14.2990   9.8388  24.1378
PFK         1.1717  25.2906  24.1189
PYK         0.0000  21.3830  21.3830
G6PDH2r    -0.0000  24.1378  24.1378
EX_ac_e     0.0000   3.8136   3.8136
EX_etoh_e   0.0000   2.2143   2.2143
EX_for_e    0.0000  11.3223  11.3223
```

![선택 반응의 FVA 구간과 pFBA 해](../.gitbook/assets/lab11/fig_fva_intervals.png)

*그림 11.3. 최적 성장의 90% 이상을 유지한다는 조건에서 선택 반응의 FVA 최소–최대 구간(회색 막대)과 pFBA 해(주황 점). PGI는 음수까지 걸쳐 방향이 바뀔 수 있고, 아세테이트·에탄올·포름산 분비 exchange는 하한 0에서 각각 최대 3.81·2.21·11.32까지 늘어날 수 있다. 폭이 0보다 큰 반응은 성장률을 해치지 않고도 flux가 달라질 수 있음을 뜻한다. 저자 계산·시각화; COBRApy 0.30.0, GLPK, fraction\_of\_optimum=0.90.*

FVA는 [blocked reaction](../glossary.md) 진단에도 쓰입니다. 최소와 최대가 모두 0인 반응은 주어진 조건에서 어떤 flux도 가질 수 없는 blocked 반응이며, 이는 흔히 [막다른 대사물](../glossary.md)이나 누락 반응의 신호입니다([Chapter 5](../chapter-5/README.md)). 위 일곱 반응은 모두 폭이 0보다 크므로 blocked가 아닙니다.
